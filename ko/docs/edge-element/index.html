엣지

엣지(Edge)는 메시를 이루는 선으로 엣지를 이동하거나 편집하면 그와 연결된 엘리먼트에 영향을 줍니다.

단축키 Shift+2 을 누르거나, 씬 상단 버튼을 통해 엣지 관련 속성들을 볼 수 있습니다.

참고

단축키 Shift+클릭드래그​

엣지를 선택하고 Shift를 누른 채로 클릭드래그를 하면, 선택된 엣지가 이동하면서 동시에 이동 방향으로 새로운 페이스가 생성됩니다.

엣지 Element 그룹 속성​

엣지를 선택했을 때, Element 그룹에 표시되는 속성들입니다.

Display​

메시의 표시 방식을 선택할 수 있는 속성입니다.

XRay Mode​

페이스를 반투명으로 표시합니다.

Hide Edge Overlay​

엣지를 보이게 하거나 숨길 수 있습니다.

Display UVSeam​

Mark Seam 툴로 설정된 엣지를 보이게 하거나 감출 수 있습니다.

참고

Mark Seam 툴은 UV 에디팅을 위한 툴로 Surface 그룹의 Mark Seam 툴을 이용하여 UV Editor 툴에서 활용할 엣지를 선택할 수 있습니다.

Select​

엣지의 선택 방식을 설정할 수 있는 속성입니다.

Backface Select​

반대편을 향하는 엣지들의 선택 여부를 설정할 수 있습니다.

Backface Select 속성이 꺼져 있을 때는, 반대편을 향하는 엣지가 선택되지 않습니다.

Backface Select 속성이 켜져 있을 때는, 반대편을 향하는 엣지도 함께 선택됩니다.

Camera Based Select​

카메라를 기준으로 현재 보여지는 엣지만 선택하는 속성입니다.

Camera Based Select 속성이 꺼져 있을 때는, 카메라에 가려진 엣지도 선택됩니다.

Camera Based Select 속성이 켜져 있을 때는, 카메라에 가려진 엣지는 선택되지 않습니다.

Border Rect Select​

페이스 가장자리 엣지만 선택하는 속성입니다.

Border Rect Select 속성이 꺼져 있을 때는 모든 엣지가 선택됩니다.

Border Rect Select 속성이 켜져 있을 때는, 페이스의 가장자리와 뚫려있는 부분의 엣지만 선택됩니다.

팁

뚫려있는 곳을 찾거나, 결합이 덜 되어있는 버텍스를 찾을 때 활용하기 좋습니다.

Soft Select​

씬(Scene)에 선택된 엣지와 인접한 엣지들을 가중치에 따라 선택하는 속성입니다.

가중치에 따라 선택한 엣지와 설정한 범위가 컬러로 표시되며, 대체로 부드러운 원형으로 표시됩니다.

Soft Selection​

Soft Selection을 사용하는 속성입니다.

Soft Selection Curve​

Soft Selection 의 모양을 결정하는 그래프 형태의 속성입니다. Soft Selection Curve 속성 옆 그래프를 누르면 편집할 수 있습니다.

Soft Selection Range​

선택한 엣지로부터 소프트셀렉션이 적용되는 범위를 설정하는 속성입니다.

유니티 기준 1m = 1.0이기 때문에, 0.5 의 경우 선택한 엣지로부터 50cm 까지 선택됩니다.

Distance​

엘리먼트를 이동시킬 때 디스플레이 되는 속성으로 이동한 거리를 나타냅니다..
