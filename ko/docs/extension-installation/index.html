ControlNet 익스텐션 설치

Stable Diffusion web UI에서는 Extension이라고 하는 여러 에드온을 설치할 수 있습니다. UModeler X의 AI 텍스처링 기능을 사용하기 위해 ControlNet이라는 Stable Diffusion web UI의 확장 기능을 설치해야 합니다.

ControlNet 설치​

  1. Extensions 탭 열기
    • web UI에서 Extensions 탭을 엽니다.
  2. URL로 설치하기
    • Extensions 탭 내에서 Install from URL 탭을 엽니다.
  3. ControlNet 저장소 URL 입력
    • URL for extension's git repository 옆 입력란에 https://github.com/Mikubill/sd-webui-controlnet.git을 입력합니다.
  4. Install 버튼 클릭
    • Install 버튼을 누르고 조금 기다립니다.
    • Installed into stable-diffusion-webui\extensions\sd-webui-controlnet. Use Installed tab to restart라는 메시지가 표시되면 설치가 완료된 것입니다.
  5. 업데이트 및 재시작
    • Installed 탭으로 이동합니다.
    • Check for updates를 클릭한 후 Apply and restart UI를 클릭합니다.

ControlNet 모델 설치​

ControlNet에는 사용할 수 있는 여러 모델이 있습니다. ControlNet 1.1에는 현재 베타 테스트 중인 14개의 모델이 포함되어 있습니다.

다운로드 및 적용​

  1. 모델 다운로드 링크 접속
    • ControlNet 1.1 모델 다운로드 페이지에 접속합니다.
  2. .pth 파일 다운로드
    • 페이지에서 파일 확장자가 .pth로 끝나는 모델 파일들을 다운로드합니다.
      • 14개의 모델의 총 용량은 약 18.8기가이며, 모든 모델을 다운로드할 필요가 없다면 선택적으로 다운로드할 수 있습니다.
      • 만약 용량이 부담이 된다면 control_v11f1p_sd15_depth.pth 만 다운로드하는 것을 권장합니다.
  3. 다운로드 받은 모델 파일 이동
    • 다운로드한 모델 파일들을 Stable Diffusion web UI가 설치된 폴더의 stable-diffusion-webui/extensions/sd-webui-controlnet/models 경로에 넣으면 모델이 적용 완료됩니다.

ControlNet과 익스텐션을 설치하는 다른 방법​

ControlNet을 다른 방법으로 설치하는 방법입니다. 이 방법을 사용하면 Stable Diffusion web UI에서 사용 가능한 여러 익스텐션의 목록을 보며 손쉽게 설치할 수 있습니다.

  1. Extensions 탭 열기
    • Extensions 탭을 열고 Available을 클릭합니다.
  2. Load from 버튼 확인
    • Load from: 옆에 표시된 주소가https://raw.githubusercontent.com/AUTOMATIC1111/stable-diffusion-webui-extensions/master/index.json인지 확인합니다.
      • 따로 수정하지 않았다면 기본적으로 위 주소로 설정되어 있습니다.
    • Load from: 버튼을 클릭합니다.
      • 설치 가능한 여러 익스텐션이 표시됩니다.
  3. ControlNet 검색
    • Search 입력 부분에 sd-webui-controlnet을 입력합니다.
  4. ControlNet 설치
    • 목록에서 sd-webui-controlnet manipulations를 찾고 설명의 오른쪽에 있는 Install 버튼을 클릭합니다.
      • 아래 이미지는 위에서 이미 설치된 상태이기 때문에 Installed로 표시되어 있습니다.
  5. 설치 확인
    • 설치가 완료되면 Installed 탭으로 돌아가 설치된 익스텐션을 확인합니다.
  6. 적용 및 재시작
    • Apply and restart UI 버튼을 누르면 설치한 익스텐션을 실제로 적용하게 됩니다.

이 방법을 통해 ControlNet 익스텐션 뿐만 아니라 여러가지 다른 익스텐션들도 손쉽게 추가할 수 있습니다.
