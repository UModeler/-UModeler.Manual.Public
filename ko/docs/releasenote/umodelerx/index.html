UModeler X Basic​

노트

UModeler X Beta 버전이 2024년 9월 13일부로 UModeler X Basic으로 변경되었습니다.

Ver 1.0.0 | 2024.9.13​

  • 수정사항
    • Grid Snap Panel : Grid Snap panel 윈도우가 개선되었습니다.
    • MultiObject Mode : Material Tool의 언두 문제를 해결했습니다.
    • Modeling Mode
      • 특정 상황에서 모디파이어 중 몇몇개가 표시되지 않는 문제를 수정했습니다.
      • Shape Tool Group : 쉐이프 툴 조작시 발생하던 크래시 에러를 수정했습니다.
      • Shape Tool Group : 높이가 있는 도형들에 음수 Height 기능이 추가되었습니다.
      • Line Tool : 오른쪽 버튼으로 해제를 할 수 있게 추가되었습니다.
      • Shrink Select Tool : 사용시 발생하던 오류를 수정했습니다.
    • Painting Mode
      • 페인팅 레이어의 혼합설정 오버레이의 작동방식이 수정되었습니다.

UModeler X Beta​

  • 수정사항

      • Insert Vertex Tool : 여러개의 Material을 사용하는 도형에서 적용된 Material이 변경되는 현상을 수정했습니다.
      • UVParam Tool : Reset UVParam 버튼명이 Reset UVParams로 수정했습니다.
      • UVParam Tool : 언두/리두시 Shift Rotation Scale값이 변하지 않는 버그를 수정했습니다.
    • Rigging Mode
      • 리깅데이터가 종종 제대로 로딩되지 않는 버그를 수정했습니다.
      • Clear Rigging Data 실행시 캐시가 제대로 삭제되지 않는 문제를 수정했습니다.
    • UVEditor
      • MultiMaterial이 제대로 표시되지 않는 버그를 수정했습니다.
    • Multi Object
      • Boolean Tool : 불린 실행 시 엣지 부분의 연결이 어긋나는것을 수정했습니다.

0.20.0 | 2024/7/29​

  • 새 기능
    • Face Color Tool : Vertex를 공유하는 옆 Face와 다른 칼라를 넣을수 있는 툴이 추가되었습니다.
      • Apply Color : 선택한 Face들에 해당 칼라를 적용합니다.
      • Select Face Colors : Face Color가 설정된 Face 들을 모두 선택합니다.
      • Reset Color : Face Color를 해제합니다.
    • UVParameter Tool : Unwrap되지 않은 Face들의 자동 생성 UV를 추가로 변경할 수 있는 툴이 추가되었습니다.
      • Shift : 자동 생성된 UV 좌표를 설정한 값으로 이동시킵니다.
      • Rotation : 자동 생성된 UV 좌표를 회전시킵니다. 0,0 을 기준으로 회전시키기 때문에 해당 자동생성이 원점에서 떨어져있는경우 회전 중심점이 다른곳으로 보일수 있습니다.
      • Scale : 자동 생성된 UV 좌표의 비율을 설정합니다.
      • Tile : 현재 선택한 Face들을 기준으로 Tiling된 UVParam를 설정합니다. 주어진 값으로 Shift Rotation Scale값을 조정합니다.
      • Select UVParams : UVParam이 설정된 Face들을 모두 선택합니다.
      • Reset UVParam : UVParam를 해제합니다.
  • 기능 향상
    • Elelemt Tool : Vertex,Edge,Face Tool 변경시 이전에 선택한 Elements 들을 기준으로 변환하여 현재 툴의 Elements를 선택하도록 변경되었습니다.
    • Overlay : 오버레이 출력방식이 변경되었습니다. 엣지의 이상출력이 수정되었습니다.
  • 수정사항
    • Mirror Modifier : 이동 기즈모의 화살표 부분 피킹이 안되던 문제를 수정했습니다.
    • Mirror Modifier : 미러 플립 사용 시 이동 기즈모가 반대로 작동하는 문제 수정하였습니다.
    • Connect Tool : 커맨터리가 표시되지 않는 버그를 수정했습니다.
    • MarkUVSeam Tool : 커맨터리가 표시되지 않는 버그를 수정했습니다.
