UModeler X Plus

1.0.5 | 2024/12/19​

  • 새 기능
    • 일반
      • Element ID Selector : ID를 통해 Element를 탐색할 수 있는 새로운 메뉴가 추가되었습니다. Tools > UModelerX > Element ID Selector를 통해 사용할 수 있습니다.
      • Repeat Tool : 사용했던 즉시 모드 툴을 다시 사용 합니다. 키보드의 ‘;’ 버튼을 눌러 사용할 수 있습니다.
  • 개선사항
    • 일반
      • UMX Snap Setting : toolbar 에서 팝업창을 통해서 사용하도록 개선되었습니다.
    • Modeling Mode
      • Element Tool : 오브젝트의 버텍스를 기준으로 다른 오브젝트에 스냅하여 회전시키는 snap rotation 기능이 추가되었습니다.
      • Relax Brush Tool : Strength 값이 최대 1.0 까지만 적용됩니다.
      • Smooth Brush Tool : Strength 값이 최대 1.0 까지만 적용됩니다.
      • Align Tool : Icon이 수정되었습니다.
  • 버그 수정사항
    • 일반
      • 생생 후 처음 저장한 오브젝트가 다시 불러올때 빌드가 새로 일어나는 버그를 수정했습니다.
      • VRChat 업로드 이후 생기는 문제를 수정했습니다.
    • Modifier
      • 몇몇 Modifier 에서 빌드가 새로 일어나는 버그를 수정했습니다.
    • Modeling Mode
      • Pivot Tool : 툴을 처음 실행할 때 기즈모가 이상한 위치에 나오는 버그를 수정했습니다.
      • Mirror Object Tool : 단축키를 통해서 Docs에 접근할 수 없는 문제를 수정했습니다.

1.0.4 | 2024/12/06​

  • 새 기능
    • MultiObject Editor
      • Mirror Object Tool : 단일 오브젝트의 Mirror Object Editor 에서도 사용 가능해집니다.
  • 개선사항
    • Modeling Mode
      • Mirror Object Tool : 미리보기가 추가되었습니다.
      • Mirror Object Tool : UV가 보존됩니다.
  • 버그 수정사항
    • 일반
      • UMX Toolbar : 설치 후 Button Icon이 보이지 않는 문제를 수정했습니다.
      • 오브젝트 복사와 씬 저장시 UV Seam, Face Color, Face Normal, UV Param 등의 데이터 중 일부가 저장되지 않는 문제를 수정했습니다.
    • MultiObject Editor
      • Pivot Tool : 설정시 모디파이어의 영향을 받는 문제를 수정했습니다.
    • Painting2D Editor
      • 페인팅 창에서 레이어 선택하지 않고 ShowWireframe 킬 경우 발생하는 에러를 수정했습니다.

1.0.3 | 2024/11/29​

  • 개선사항
    • 일반
      • UModelerize에서 멀티 UV 채널을 지원합니다.
    • Modeling Mode
      • Weld Tool : UV가 보존될 수 있도록 수정했습니다.
    • UV Editor
      • UVSelect Tool : 드래그 시작점이 UVFace 일 경우 쉬프트나 컨트롤을 누른 상태에서 해당 페이스 이동이 아닌 Rect Select 기능이 작동하도록 수정했습니다.
  • 버그 수정사항
    • 일반
      • EdMesh Info : 메모리 가비지 콜렉션 이슈를 수정했습니다.
    • Modeling Mode
      • Eraser Tool : Unwrap 되지 않은 모델에서 작동 시 모델이 사라지는 현상을 수정했습니다.
      • Bridge Tool : Unwrap 되지 않은 모델에서 작동하지 않는 현상을 수정했습니다.
    • UV Editor
      • 멀티 UV Channel 저장시 문제되는 부분을 수정했습니다.
      • UV Channel을 바꿀때 Island Boundary 갱신이 늦는 버그를 수정했습니다.
      • UV Channel에서 UVIsland 갱신시 생기는 버그를 수정했습니다.
      • UVFace Tool : UV선택 취소시 잔여 루프가 남는 문제를 수정했습니다.
      • Unwrap Tool Group : 언랩 할 때 현재 UV 채널 생성 상태를 체크합니다.
      • UVIsland Tool : 현재 ****Material 에 상관없이 모든 UVIsland가 선택되던 버그를 수정했습니다.
      • Auto Layout Tool : 멀티 UV Channel 에서 문제 생기는 이슈를 수정했습니다.
      • Sew Tool : 멀티 UV Channel 에서 문제 생기는 이슈를 수정했습니다.
      • Move Sew Tool : 멀티 UV Channel 에서 문제 생기는 이슈를 수정했습니다.

1.0.2 | 2024/11/26​

  • 개선사항
    • 일반
      • UMX Settings Toolbar : 씬을 로딩 할 때 켜진 상태로 설정됩니다.
      • 저장 데이터를 선택했을때 인스펙터에 나오는 바이트스트림으로 인한 속도 저하를 개선했습니다.
      • UModelerXEditableMesh를 직접적으로 AddComponent하는 것을 막았습니다. 해당 작업은 문제를 일으킬 위험이 있습니다.
    • Modeling Mode
      • Collapse Tool : 실행 시 UV 가 보존됩니다.
      • Merge Tool : 실행 시 UV 가 보존됩니다.
      • Bridge Tool : 실행 시 UV 가 보존됩니다.
      • Bevel Tool : 이전에 사용했던 Width 값이 보존됩니다.
      • Pivot
    • Rigging Mode
      • Envelope Tool : Root 이상의 본에서는 캡슐이 표시되지 않습니다.
    • Pen Pressure
      • Brush UI 에서 Pen 토글이 표시됩니다.
  • 버그 수정사항
    • 일반
      • 유니티 빌드시 문제생기던 문제를 해결했습니다.
      • 스크립트 리로딩시 발생하는 선택 상태의 싱크 워닝 문제를 수정했습니다.
      • 프로퍼티 리로딩 워닝 발생을 수정했습니다.
    • Rigging Mode
      • Envelope Tool
        • 캡슐의 범위가 본에 맞게 표시됩니다.
        • 캡슐의 크기 변형 작업이 Undo/ Redo 되지 않는 현상을 수정했습니다.
    • UV Editor
      • 멀티채널 UV를 켜고 끌때 몇몇 경우 UV Unwrap상태가 갱신이 안되던 버그를 수정했습니다.
      • Plane Unwrap Tool : 옆면 언랩시 Y축 기준으로 정렬되도록 수정했습니다.
    • Hotspot Editor
      • 핫스팟 에디터의 몇몇 아이콘이 작은 화면에서 뜨지 않던 문제를 수정했습니다.
      • 핫스팟과 UV에디터가 떠있는 경우 배경이 깜빡거리는 현상을 수정했습니다.
    • Export
      • 일부 환경에서 Export 시간이 오래 걸리는 현상을 수정했습니다.

1.0.0 | 2024/11/20​

  • 새 기능
    • Hotspot Layout
      • 핫스팟 레이아웃이 추가되었습니다. 미리 설정된 레이아웃에 맞춰서 선택한 페이스들의 UV를 자동으로 배치하거나 빌드와 동시에 UV 언랩되지 않은 페이스들을 레이아웃에 맞춰 UV를 배치해줍니다.
    • Modeling Mode
      • Separate Parts Tool : 선택한 오브젝트의 연결되지 않은 부분들을 떼어내 새로운 오브젝트들로 생성하는 툴입니다.
      • Follow Tool : 지정한 경로를 따라서 프로파일(면)을 이동시켜 새로운 3D 모델을 생성하는 툴입니다.
    • Rigging Mode
      • Human Avatar Tool : 휴먼 타입 아바타를 설정해주는 툴이 추가되었습니다. Aniamtion Root에 Animator를 추가하고 해당 Aniamtor에 휴먼타입 아바타를 추가합니다.
    • UVEditor
      • Multiple UV Channel
        • 멀티채널 UV가 추가되었습니다. 0번 채널이 아닌 다른 채널의 UV를 에디트 하는 기능입니다. 0번아 아닌 채널을 사용하기 위해선 해당 채널을 선택한 후 활성화하는 작업이 필요합니다.
    • Delete UMX Component : Current Scene, Prefabs 또는 Selected Object에서 UMX Component를 제거합니다.
  • 개선사항
    • 일반
      • Gameobject : 새로 생성되는 오브젝트와 모디파이어의 명명 규칙이 유니티와 동일하게 적용되었습니다.
      • Object Menu : 하이어라키창을 우클릭을 했을 때 나오는 메뉴의 3D Object > UModelerX를 통해 새로운 UModelerX 오브젝트를 생성할 수 있습니다.
      • Preferences
        • Preferences 의 Ed Mesh 가 Editable Mesh와 Modeling, Rigging으로 나누어졌습니다.
        • Editable Mesh에 Save data in the scene folder 옵션이 추가되었습니다.
        • Hotspot Editor가 추가되었습니다.
        • Debug 카테고리에 PrefabPath 가 추가됩니다. Delete UMX Component 실행 시 Prefab 데이터를 찾는 위치를 지정할 수 있습니다.
      • Export
        • Export 시간이 단축됩니다.
        • Export 파일 생성 시 소수점 구분 기호로 ‘.’ 를 사용합니다.
    • Modeling Mode
      • DisplayMode : Xray Mode가 Display Mode로 통합되고 Checkered Mode가 추가되었습니다.
        • Checkered Mode는 Material의 형태를 Checkered Material로 변경합니다.
    • Rigging Mode
      • Overlay Animation : 현재 본을 기준으로 애니메이션 된 상태의 오버레이를 출력합니다.
      • Symmetry Mode(Only Bone) : 반대편에 Left Right를 제외한 같은 본 이름이 있는 경우 함께 선택해 컨트롤할 수 있습니다.
      • Rigging Tool Group이 분리되었습니다.
        • Weight Gruop
          • 각 버텍스의 가중치를 수정하는 툴 그룹입니다.
        • Bone Group
          • 본을 추가/삭제하거나 포즈 설정에 관련된 툴 그룹입니다.
      • Weight Mirror Tool : 입력창에 좌우 관련 워드를 없애고 L,R,Left,Right 기준으로 좌우 본을 분류합니다.
    • UI
      • Commentary : 경고 로그가 표시되지 않습니다.
      • EdMeshInfo : SceneView의 EdMeshInfo Panel이 조금 더 작아지고 원하는 정보를 켜고 끌 수 있게 되었습니다.
      • UMX Setting Toolbar
        • Unity 2021.1 이후 : SceneView에서 사용할 수 있는 전용 툴바가 추가되었습니다. OverlayMenu를 통해 접근할 수 있습니다.
        • Unity 2020.3 이전 : Editor 좌상단에 UMX Display Settings 버튼이 추가되었습니다.
    • 기타
      • 기존 Line Tool이 Add Group으로 이동되면서 Knife Tool로 변경되었습니다.
  • 버그 수정사항
    • 일반
      • UModelerize시 일부 모델에서 정상작동 안되는 버그를 수정했습니다.
      • 오브젝트 복사시 Mirror Modifier 에서 빌드가 일어나지 않는 버그를 수정했습니다.
    • Rigging Mode
      • Envelope Tool : Show Handler 기능이 정상적으로 작동하지 않는 현상을 수정했습니다.
