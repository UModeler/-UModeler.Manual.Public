UModeler Hub

1.0.1 | November 19, 2024​

  • 로그인 화면에서 계정 생성에 관련한 문구가 추가되었습니다.
  • Plus 사용자를 위한 사용자 정보 수집에 대한 Opt-in/Opt-out 설정 페이지가 추가되었습니다.

1.0.0 | September 14, 2024​

  • 구글 회원가입 및 로그인 추가
  • UModeler X Plus 제공
  • 인보이스 번호 검증 기능 구현

0.12.14 | July 2, 2024​

  • Unity6의 ProBuilder가 설치된 프로젝트에 설치했을 때 오류가 발생하는 문제를 수정했습니다.

0.12.13 | June 27, 2024​

  • 유니티 빌드 : 유니티 빌드시 2021 이하 버전에서 문제가 생겨서 원상태로 되돌렸습니다.

0.12.12 | June 27, 2024​

  • 유니티 빌드 : 유니티 빌드시 불필요한 UModelerHub 실행 데이터가 들어가는 것을 줄였습니다.
  • 로그인 관련 데이터가 오류나는 버그를 수정했습니다.

0.12.11 | 2024/5/29​

  • Unity 2023 버전에서 나오는 워닝 메시지를 삭제했습니다.

0.12.10 | 2024/4/17​

  • 유니티 2023.2와 이후 버전에서 로그인시 생기는 오류를 수정했습니다.

0.12.9 | 2024/4/12​

  • 프로빌더가 있을 경우 로그인 툴바를 아래로 내렸습니다.
  • UModelerX를 업데이트 하고 나서 경고메시지가 나타나는 현상을 수정하였습니다.
  • UModeler Hub용 Resources 폴더를 빌드 영향을 받지 않도록 Editor 밑으로 옮겼습니다.

0.12.8 | 2024/2/21​

  • 로그인 시 2023.x 버전에서 발생하던 에러를 수정했습니다.
  • Unity Play시 발생하는 Warning 메시지를 수정했습니다.

0.12.7 | 2024/2/6​

  • 스크립트 리로딩이 일어났을때 허브 윈도우가 뜨는 버그를 수정했습니다.
  • UModelerHub Preference 의 Launch On Start를 기본으로 비활성화 했습니다.

0.12.6 | 2024/1/9​

  • 유니티 에디터에서 Play/Stop 을 진행할때 생기는 에러를 수정했습니다.

0.12.5 | 2023/12/23​

  • Unity 의 Play버튼을 눌렀을때 허브창이 뜨는 버그를 수정했습니다.

0.12.4 | 2023/12/12​

  • UModeler X 업데이트나 핫리로드 후 로그인 상태가 풀리는 이슈 처리
  • 회원 가입 중 이메일 인증이 안되는 이슈 처리
  • 회원 가입 절차에서 패스워드 재입력 과정 생략

0.12.3 | 2023/11/10​

  • 허브만 설치된 상태에서 기즈모 혹은 핸들이 씬뷰에 안보이는 이슈를 해결함.

0.12.2 | 2023/11/07​

  • UModeler X 패키지 폴더의 manifest.json과 package-lock.json에 대한 버전 설정을 절대 경로에서 상대 경로로 변경했습니다
  • Unity 2023.x의 로그인 또는 가입 페이지에서 이메일, 비밀번호 등을 입력할 때 발생한 텍스트 편집 오류를 수정했습니다.

0.12.1 | 2023/10/6​

  • 가입시 발생하는 Conflict(409) 오류에 대한 메세지가 구체적으로 표시됩니다.

0.12.0 | 2023/9/21​

  • 좀더 쉽게 로그인할 수 있도록 레이아웃이 개선되었습니다.

0.11.1 | 2023/09/04​

  • 가입시 일부 오류가 발생하는 부분을 수정하였습니다.

0.11.0 | 2023/09/04​

  • 초대 기능이 제거되었습니다.
  • 유니티 버전이 권장 버전이 아닐경우 상단에 경고문이 표시됩니다.
  • 허브 창 제목에 버전이 표시됩니다.
  • Preference 의 허브 설정 화면이 스크롤되지 않는 현상을 수정하였습니다.

0.10.4 | 2023/08/25​

  • “Keep me signed in”이 설정되어있음에도 불구하고 유니티 에디터를 닫고 다시 열었을때 인증이 유지되지 않는 현상을 수정하였습니다.

0.10.3 | 2023/8/21​

  • 애셋 페이지에서 애셋이 다운로드 되지 않는 현상을 수정하였습니다.

0.10.2 | 2023/08/16​

  • 전반적으로 안정성이 향상되었습니다.
  • 사용자 가입시 에러 메세지가 정상적으로 표시되지 않는 현상을 수정하였습니다.

0.10.1 | 2023/07/19​

  • 특정 상황에서 로그인이 되지 않는 부분을 수정하였습니다.

0.10.0 | 2023/06/28​

  • 회원 가입과 Account 페이지에 국가 선택 UI가 추가 되었습니다.
  • 디스코드 링크 URL이 변경되었습니다.
  • 친구 초대를 위한 초대탭이 추가되었습니다.
  • 사용 중 로그인 유지가 안되는 이슈가 해결되었습니다.
  • 한글로 표시되던 메세지를 영어로 변경하였습니다.
  • 프록시 설정 기능 추가

0.9.11 | 2023/05/09​

  • 공통 라이브러리 변경점을 적용한것 외에는 변화가 없습니다.

0.9.7 | 2023/05/03​

  • 특정 상황에서 업데이트 정보를 가져오지 못해서 발생한 아래의 이슈를 해결하였습니다.
    • 위치 : Tripolygon.UModeler.Hub.Notifications.UpdateHubNotifier+UpdateHubNotification
    • 오류명 : System.ArgumentNullException: Value cannot be null.

0.9.6 | 2023/04/19​

  • 자동 로그인 관련 이슈가 처리되었습니다.

0.9.5 | 2023/04/17​

  • 전반적으로 안정성이 향상되었습니다.

0.9.4 | 2023/04/07​

  • 허브 창을 띄울때 간혹 발생하는 예외를 수정하였습니다.
  • 한글로 출력되던 오류, 경 메세지 대부분을 영문으로 변경되었습니다.

0.9.3 | 2023/04/04​

  • 계정 정보 페이지에서 간혹 FullName 이 표기되지 않는 문제점을 수정하였습니다.

0.9.2 | 2023/04/01​

  • UModeler Hub와 Zepeto Studio의 충돌점이 발견되었습니다. Zepeto Studio → Export as .zepeto 가 동작하지 않는 버그입니다. UModeler Hub 0.9.2 버전을 받아주세요.
    • 기존 버전의 UModeler Hub를 설치하신 유저분들은 꼭 0.9.2로 업데이트 하시고 혹시라도 Assets/UModeler-Hub/package.json 파일이 있다면 삭제해주세요.
