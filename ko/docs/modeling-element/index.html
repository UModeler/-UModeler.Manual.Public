Element 그룹

Element 그룹은 모델링 모드 최상단에 위치하며 각 엘리먼트(Vertex, Edge, Face, Object)에 관한 속성이 모여 있는 그룹입니다.

편집할 엘리먼트 변경​

모델링 모드의 원본 메시 Source Mesh는 엘리먼트 선택 후 편집이 가능합니다.

3가지 편집 가능한 엘리먼트(Vertex, Edge, Face)와 편집 상태에서 빠져나오는 Object가 있으며 씬(Scene) 상단 아이콘들을 통해 어떤 엘리먼트로 편집할지 선택할 수 있습니다.

엘리먼트의 선택에 따라 Element 그룹 내에 공통적으로 적용되는 속성과 각 엘리먼트에 따라 다르게 적용되는 속성이 있습니다.

아래 각 엘리먼트에 대한 페이지를 통해 각 엘리먼트에 대한 설명과 엘리먼트 선택에 따른 속성을 쉽게 확인할 수 있습니다.

단축키 Shift+클릭드래그​

각 엘리먼트(버텍스, 엣지, 페이스)를 선택하고 나서 Shift+클릭드래그를 이용하여 엘리먼트를 이동시키면, 이동한 방향으로 선택한 엘리먼트가 돌출되거나 밀어내듯이 새로운 엘리먼트를 생성할 수 있습니다.

참고

페이스 엘리먼트의 경우 페이스 Element 그룹 속성에 있는 Extrude Option 속성으로 페이스가 어떻게 돌출되는지 선택할 수 있습니다.

공통 속성​

Element 그룹에서 각 엘리먼트에 공통적으로 적용되는 속성들입니다.

Display​

메시의 표시 방식을 선택할 수 있는 속성입니다.

XRay Mode​

페이스를 반투명으로 표시합니다.

팁

XRay Mode 속성을 이용하면 복잡한 모델의 뒷면에 있는 엘리먼트들을 손쉽게 확인할 수 있습니다.

Hide Edge Overlay​

엣지를 보이게 하거나 숨길 수 있습니다.

Select​

엘리먼트의 선택 방식을 설정할 수 있는 속성입니다.

Backface Select​

반대편을 향하고 있는 엘리먼트의 선택 여부를 설정할 수 있습니다.

Backface Select 속성이 꺼져 있을 때는, 반대편을 향하는 엘리먼트가 선택되지 않습니다.

Backface Select 속성이 켜져 있을 때는, 반대편을 향하는 엘리먼트도 함께 선택됩니다.

Camera Based Select​

카메라를 기준으로 현재 보여지는 엘리먼트만 선택하는 속성입니다.

Camera Based Select 속성이 꺼져 있을 때는, 카메라에 가려진 엘리먼트도 선택됩니다.

Camera Based Select 속성이 켜져 있을 때는, 카메라에 가려진 엘리먼트는 선택되지 않습니다.

Soft Select​

씬(Scene)에 선택된 엘리먼트와 인접한 엘리먼트를 가중치에 따라 선택하는 속성입니다.

가중치에 따라 선택한 엘리먼트와 설정한 범위가 컬러로 표시되며, 대체로 부드러운 원형으로 표시됩니다.

Soft Selection​

Soft Selection을 활성화하는 속성입니다.

Soft Selection Curve​

Soft Selection의 가중치 모양을 결정하는 그래프 형태의 속성입니다. Soft Selection Curve 속성 옆 그래프를 누르면 편집할 수 있습니다.

Curve 형태 예시

Curve 형태 별 선택 모양 변화 예시입니다.

Soft Selection Range​

선택한 엘리먼트부터 소프트셀렉션이 적용되는 범위를 설정하는 속성입니다.

유니티 기준 1m = 1.0이기 때문에, 0.5 의 경우 선택한 엘리먼트로부터 50cm 까지 선택됩니다.

버텍스 스냅핑​

V키를 누르고 기즈모(핸들)을 움직이면 선택된 엘러먼트를 마우스가 가리키는 버텍스의 위치로 움직이거나 혹은 그 위치로 엘러먼트의 위치를 정렬시킬 수 있습니다.

스냅 로테이션​

마우스로 버텍스를 선택한 뒤 드래그하여 피벗을 기준으로 오브젝트를 회전시킬 수 있습니다. 회전은 마우스에 가장 가까운 다른 버텍스 방향을 바라보도록 설정됩니다.

회전하는 정도를 GUI를 통해 확인할 수 있습니다.

줄자 (Ruler)​

선택된 엘러먼트의 길이를 표시합니다. 버텍스를 선택하면 선택된 순서대로 버텍스 사이 거리가 표시됩니다. 엣지는 선택된 모든 엣지의 길이가 표시되며 페이스는 선택된 페이스를 구성하는 모든 엣지의 길이가 표시됩니다. 씬 뷰의 우측 하단에는 총 길이의 합(Total Length)이 표시됩니다.

줄자는 Preferences > UModeler X > Display > Ruler 속성을 활성화/비활성화 시킬 수 있습니다.
