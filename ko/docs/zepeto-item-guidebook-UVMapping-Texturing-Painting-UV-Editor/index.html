UV 에디터

UV 매핑은 2D 펼친 맵에 해당합니다. 3D 객체에 텍스처와 색상을 적용하기 위해 UV라고 불리는 2D 좌표를 사용할 수 있습니다.

이 작업은 Editor -> UVEditor로 이동하여 수행할 수 있습니다.

Top Menu 소개​

UV 에디터 인터페이스는 씬 뷰와 매우 유사하며, 비슷한 단축키를 공유합니다.

상단의 아이콘을 사용하거나 Space Bar를 사용하여 화면을 탐색할 수 있으며, 각각 W, E, R 키를 사용하여 이동, 회전, 크기 조정을 할 수 있습니다.

사각형 도구를 사용하여 위치와 크기를 사각형 모서리를 사용하여 조정할 수도 있습니다.

측면의 아이콘은 UV Vertex, UV Edge, UV Face를 나타내며, 마지막 아이콘은 연결된 UV 엘리멘트를 선택하는 아이콘입니다.

기어 아이콘은 설정을 위한 것입니다. 여기에서 스냅 및 기타 UV 에디터 구성 설정을 수정할 수 있습니다.

중앙 창은 선택된 객체의 UV 값을 표시합니다. 이 창에서 UV를 배치하여 텍스처 작업을 수행할 수 있습니다.

Unwrap​

Unwrap 툴은 3D 객체를 2D 평면에 투영합니다. 3D 객체의 면을 선택하고 필요한 언래핑을 실행하십시오.

Plane​

Plane 툴은 평면 투영을 위해 사용됩니다. 각 축을 따라, 월드 축 방향으로 또는 면의 평균 노멀 방향으로 투영할 수 있습니다.

Cube​

Cube 툴은 선택된 면이 향하는 방향(x, y, z)을 기준으로 UV를 펼칩니다.

Auto Layout​

Auto layout 툴은 선택된 면을 자동으로 펼칩니다.

Cancel​

Cancel 툴은 펼친 UV를 초기화합니다.

LSCM​

LSCM 툴은 모델링 툴의 MARK UV Sceam과 UNMARK UV Sceam 도구를 사용하여 UV 엣지를 임의로 잘라내고, 이러한 잘린 엣지를 따라 UV를 펼칩니다.

원하는 엣지를 선택하고 MARK UV Seam을 클릭하여 UV 솔기를 생성합니다. 제거하려면 엣지를 다시 선택하고 UNMARK UV Seam을 클릭합니다.

이 방법을 사용하여 솔기를 설정할 면을 선택한 후 LSCM을 클릭합니다. 그러면 UV가 표시된 솔기를 따라 펼쳐집니다.

Selection​

All​

All 툴은 선택된 면의 모든 요소를 선택합니다.

Invert​

Invert 툴은 선택 영역을 반전시킵니다.

Loop​

Loop 툴은 선택된 엣지에 연결된 엣지를 선택합니다.

Shared​

Shared 툴은 UV 공간에서는 분리되어 있지만 3D 공간에서는 공유되는 UV 엘리멘트를 선택합니다.

Quick Transform​

Flip Horizontal​

Flip Horizontal 툴은 선택된 엘리멘트를 수평으로 뒤집습니다.

Flip Vertical Tool​

Flip Vertical 툴은 선택된 엘리멘트를 수직으로 뒤집습니다.

Flip Center Tool​

Flip Center 툴은 선택된 엘리멘트를 수평 및 수직으로 뒤집습니다.

+90​

+90 툴은 선택된 엘리멘트를 시계 방향으로 90도 회전시킵니다.

-90​

-90 툴은 선택된 엘리멘트를 반시계 방향으로 90도 회전시킵니다.

Alignment​

Auto​

Auto 툴은 각 엘리멘트를 중앙에 정렬합니다.

Quad​

Quad 툴은 UV를 직사각형 모양으로 펼칩니다.

Boundary​

Boundary 툴은 마지막으로 선택된 엘리멘트의 외곽선을 정렬하고 크기를 맞춥니다.

Left​

UV 에디터의 Left 툴은 선택된 UV 엘리멘트를 왼쪽으로 정렬합니다.

Right​

UV 에디터의 Right 툴은 선택된 UV 엘리멘트를 오른쪽으로 정렬합니다.

Top​

UV 에디터의 Top 툴은 선택된 UV 엘리멘트를 위쪽으로 정렬합니다.

Center V​

UV 에디터의 Center V 툴은 선택된 UV 엘리멘트를 수직 중앙으로 정렬합니다.

Bottom​

UV 에디터의 Bottom 툴은 선택된 UV 엘리멘트를 아래쪽으로 정렬합니다.

Space H​

UV 에디터의 Space H 툴은 선택된 UV 버텍스를 수평으로 균등하게 정렬합니다.

Space V​

UV 에디터의 Space V 툴은 선택된 UV 버텍스를 수직으로 균등하게 정렬합니다

.

Weld And Break​

Sew​

Sew 툴은 UV 에디터에서 공유된 UV 엣지를 연결합니다. 공유된 엣지를 선택하거나 공유 툴을 사용하여 선택한 다음, Sew를 사용하여 병합합니다.

Move Sew​

Move Sew 툴은 공유 엣지가 병합될 때 연결된 엘리멘트를 함께 이동시킵니다.

Weld 툴은 UV 버텍스를 병합합니다.

Weld To First​

Weld To Ffirst 툴은 처음 선택한 버텍스의 위치로 병합합니다.

Weld To Center​

Weld To Center 툴은 선택된 버텍스의 중간 위치로 병합합니다.

Weld To Last​

Weld To Last 툴은 마지막으로 선택된 버텍스의 위치로 병합합니다.

Detach​

Detach 툴은 선택된 면을 연결된 UV 엘리멘트에서 분리합니다.

Arrange​

Arrange 그룹 기능은 UV 엘리멘트를 텍스처 공간에 균일하게 배치합니다.

Pack​

Pack 툴은 UV를 텍스처 공간에 겹치지 않게 배치하고 빈 공간을 채웁니다.

Fit​

Fit 툴은 이미 배치된 UV를 텍스처 공간에 꽉 채워 배치합니다.

Misc​

Export​

Misc 그룹 아래의 Export 툴은 UV 레이아웃을 이미지 파일로 내보냅니다.

내보낸 이미지는 Photoshop이나 Paint 같은 소프트웨어에서 이미지 편집의 가이드라인으로 사용할 수 있습니다.
