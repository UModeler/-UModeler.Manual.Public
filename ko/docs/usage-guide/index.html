사용 가이드

이 가이드는 AI 텍스처링을 사용하기 위한 기초적인 단계를 설명합니다. 먼저, AI 텍스처링을 활용하기 전에 필요한 설정 작업을 진행하겠습니다.

연동을 위한 배치파일 편집​

UModeler X의 AI 텍스처링 기능과 Stable Diffusion web UI를 연동하기 위해서는 먼저 API 설정이 필요합니다. 아래 단계를 통해 배치 파일을 수정하여 API 사용이 활성화됩니다.

  1. webui-user.bat 파일을 우클릭하여 '편집'을 선택합니다.

  2. set COMMANDLINE_ARGS= 라인을 찾아 옆에 --api를 추가합니다.

  3. 아래와 같이 표시되면 정상입니다.

    /stable-diffusion-webui/webui-user.bat
    @echo off

    set PYTHON=
    set GIT=
    set VENV_DIR=
    set COMMANDLINE_ARGS=--api

    call webui.bat

  4. 파일을 그대로 저장하고 창을 닫습니다.

사용 순서​

  1. Stable Diffusion web UI가 실행 중인 상태에서만 해당 기능을 사용할 수 있습니다.

    • 설치되어 있지 않은 경우, 아래 설치 가이드를 따라 설치합니다.

      📄️ 설치 가이드

  2. Stable Diffusion web UI가 설치된 폴더로 이동하여, webui-user.bat 파일을 실행합니다.

    • 실행 시 cmd 창이 뜨며, 이 창은 AI 텍스처링 기능 사용 중에는 닫으면 안 됩니다.
    • 해당 창에서 진행 상황을 확인할 수 있습니다.

  3. UModeler X에서 UV 언랩이 된 모델을 선택합니다.

  4. 페인팅 모드를 클릭하고, Start Painting 버튼을 클릭하여 페인팅 모드를 시작합니다.

  5. Paint 레이어를 생성하거나, 기본으로 생성된 Paint 레이어를 클릭하여 선택합니다.

  6. Properties 영역에 있는 AI 텍스처링을 선택합니다.

  7. 표시된 여러 속성 중에 Use Depth Control Net 속성을 클릭하여 체크합니다.

  8. ControlnetModel 옆 드롭다운 메뉴에서 control_v11f1p_sd15_depth로 선택합니다.

    • ControlNet의 버전에 따라 이름이 다를 수 있으나, 이름 중 가장 우측의 단어가 'depth'인 옵션을 선택합니다.

  9. 원하는 이미지의 내용을 위에 있는 Prompt 입력 부분에 입력합니다.

  10. 아래에 있는 Generate 버튼을 클릭합니다.

    • 진행 상황은 아래의 진행 표시줄과 cmd 창에서 확인할 수 있습니다.

  11. 이미지 생성이 완료되면, 아래와 같은 메시지가 표시됩니다.

  12. OK 버튼을 클릭하면 Result 영역에 생성한 이미지가 표시됩니다.

    • 이미지를 선택하고 Apply Texture 버튼을 클릭합니다.

  13. 텍스처가 모델 메시에 반영됩니다.

  14. 만약 원하는 이미지가 아닌 경우, 9번부터 다시 진행합니다.

    • 이때 프롬프트 Prompt 내용을 수정합니다.
