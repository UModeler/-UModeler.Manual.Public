파일 관리

파일 가져오기 및 변환​

외부 소스에서 모델링 또는 텍스처 파일을 가져오려면 Project 패널의 원하는 폴더로 드래그 앤 드롭할 수 있습니다.

가져온 모델을 보려면 Hierarchy에 드래그 앤 드롭하십시오.

외부 모델은 직접 편집할 수 없습니다.

이 모델을 UModeler X로 수정하려면 객체를 오른쪽 클릭하고 UModeler X - UModelerize를 선택하십시오.

파일 저장​

Unity에서는 일반적으로 외부 소스에서 모델링 파일을 가져와 장면에서 사용합니다. 따라서 UModeler X로 만든 모델은 장면 내에서만 존재할 수 없습니다.

UModeler X로 생성된 모델을 포함하는 Scene 파일을 저장하면 UModeler X 파일과 모델링 파일이 자동으로 Scene 파일이 위치한 폴더에 Scene 파일 이름으로 저장됩니다.

더 쉽게 관리하려면 각 항목에 대한 개별 폴더를 만들고 항목 이름으로 장면을 지정하는 것이 편리합니다.
