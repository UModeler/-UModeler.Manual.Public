UModeler X UI

UModeler X는 세 가지 주요 모드: Modeling 모드, Rigging 모드, Painting 모드를 가지고 있습니다.

Inspector 패널에서 처음 열면 기본적으로 Modeling 모드로 설정됩니다.

Modeling 모드​

Modeling 모드에서는 도형 생성 및 수정 작업을 수행합니다.

원시 도형 또는 도면을 선택한 후 드래그하여 기반을 생성하고 마우스를 움직여 객체의 높이를 설정합니다.

이렇게 생성된 객체는 Tool Properties에서 값을 변경하여 수정할 수 있습니다.

Basics of modeling

Painting 모드​

Painting 모드에서는 UV가 생성된 객체에 색상을 추가할 수 있습니다.

레이어를 추가하여 원하는 색상을 그리거나 텍스처를 로드할 수 있습니다.

Scene view에서 브러쉬를 사용하여 페인팅하거나 2D 평면에 그림을 그릴 수 있습니다.

Painting

Check

Rigging 모드​

Rigging 모드는 캐릭터나 객체에 뼈대를 리깅하고 객체가 뼈대의 움직임에 따라 움직이도록 할당하는 것을 포함합니다.

도구를 사용하여 뼈대를 생성하거나 모델링을 뼈대에 할당할 수 있습니다.

Rigging

Check

UV Editor​

UV는 3D 객체 표면의 2D 레이아웃을 말하며, 이를 통해 색상이나 재료를 적용할 수 있습니다.

UV Editor는 이러한 UV 맵을 생성하고 수정할 수 있는 곳입니다.

UV Editor

Check

Painting 2D Editor​

Painting 2D 에디터 페인팅 모드일 때 3D 뷰 대신 2D 모드에서 페인팅할 수 있게 합니다.
