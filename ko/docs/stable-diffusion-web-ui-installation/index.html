Stable Diffusion web UI 설치

Stable Diffusion web UI는 스테이블 디퓨전을 사용하기 쉽게 해주는 브라우저 인터페이스입니다. 웹 기반의 인터페이스로 사용이 간편하고, 여러 편의 기능이 포함되어 있습니다.

Git을 통한 Stable Diffusion web UI 설치​

  1. Git Bash 또는 PowerShell 사용하기

    • 설치를 원하는 폴더 내의 빈 공간을 우클릭한 후 'Open git bash here'나 '여기에 PowerShell 창 열기'를 선택합니다.

  2. 새로 열린 창에 아래의 명령어를 입력합니다.

    git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git

  3. 명령어를 입력하고 Enter를 누르면 설치가 자동으로 진행됩니다.

    정보

    git clone​

    • git clone 명령어는 Git을 사용하여 원격 저장소의 복사본을 로컬 컴퓨터에 생성하는 명령어입니다.

  4. 설치 확인

    • 설치가 완료된 후 원하는 폴더 내에 'stable-diffusion-webui'라는 폴더가 생성되었다면 설치가 정상적으로 완료된 것입니다.

Stable Diffusion web UI 실행​

  1. 웹 UI 실행 파일 찾기
    • 설치된 폴더 내에서 webui-user.bat 파일을 찾습니다. 이 파일은 Web UI를 구동하기 위한 배치 파일입니다.
  2. 파일 실행
    • webui-user.bat 파일을 실행합니다.
  3. 자동 설치 프로세스 확인
    • 실행하면 새로운 CMD 창이 열리고, 필요한 여러 파일과 라이브러리가 자동으로 설치됩니다.

    • 설치가 완료되면 웹 브라우저를 열고 http://127.0.0.1:7860/ 주소로 접속합니다.
