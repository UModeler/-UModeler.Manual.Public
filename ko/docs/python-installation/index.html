Python 설치

Stable Diffusion web UI를 실행하기 위해서는 Python을 설치해야 합니다.

Python 설치 파일 다운로드​

Python 3.10.6을 설치해야 합니다. 아래의 링크를 통해 해당 버전을 다운로드받을 수 있는 페이지로 이동할 수 있습니다.

  • Python 3.10.6 다운로드 페이지

해당 웹페이지에 접속한 후 아래로 스크롤하면 'Files'라고 볼드체로 표시된 섹션을 찾을 수 있습니다.

'Files' 섹션에서는 여러 버전의 파이썬 설치 파일이 나열되어 있는데, 여기서 'Windows installer (64-bit)' 링크를 클릭하여 다운로드합니다.

Python 설치​

  1. 다운로드한 Python 설치 파일을 실행하여 설치를 시작합니다.
  2. 설치 화면이 열리면, 'Add Python 3.10 to PATH' 옵션을 반드시 체크하여야 합니다.
    • 이 옵션은 Python을 시스템의 PATH 환경 변수에 추가하게 되며, 이후에 명령 프롬프트나 터미널에서 python 이라는 명령어를 직접 실행할 수 있게 됩니다.
  3. 나머지 설치 과정은 화면의 지시에 따라 진행합니다.
  4. 설치가 완료되면 'Disable path length limit' 버튼을 클릭합니다.
    • 이 옵션은 Windows의 파일 경로 길이 제한을 해제하는 기능입니다.
      • Windows에서는 기본적으로 파일의 전체 경로(디렉토리 포함)가 260자를 넘지 않도록 제한하고 있습니다.
      • Python 프로젝트에서는 종종 이런 제한을 초과하는 긴 경로를 사용해야 하는 경우가 있으므로, 이 제한을 해제하는 것이 좋습니다.

Python 설치 확인​

Python 설치가 성공적으로 완료되었는지 확인해야합니다. 이전 Git 설치를 확인했던 방식과 동일하며, 이번에는 명령 프롬프트에 'git' 대신 'python'을 입력해야 합니다.

  1. 입력 창 열기

    • 바탕화면이나 파일 탐색기에서 원하는 위치를 선택하고 마우스 오른쪽 버튼을 클릭합니다.
    • 나타나는 드롭다운 메뉴에서 'Open git bash here'를 선택합니다.
    • PowerShell
      1. 바탕화면이나 파일 탐색기의 원하는 위치에서 Shift+우클릭합니다.
      2. 드롭다운 메뉴에서 '여기에 PowerShell 창 열기'를 선택합니다.

  2. 새로 열린 콘솔 창 혹은 터미널 창에서 python --version을 입력한 후 Enter를 누릅니다.

  3. 화면에 'python 3.10.6'이라는 버전 정보가 정확히 출력된다면, Python의 설치가 정상적으로 완료된 것입니다.

    • 다른 버전 정보가 출력되거나, Python에 관련된 정보가 전혀 나오지 않는 경우, 설치 과정에서 문제가 발생했을 수 있으므로 다시 설치를 시도해야 합니다.
