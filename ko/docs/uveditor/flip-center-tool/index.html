Flip Center 툴

UV 에디터의 Flip Center 툴은 선택한 UV 엘리먼트의 중심을 기준으로 위치를 상하와 좌우 반전하는 툴로, 결과적으로 대각선으로 반전이 되는 툴입니다.

UV 에디터의 커서가 활성화되어 있을 경우는 커서의 위치를 중심으로 하여 중앙 반전합니다.

참고

UV 에디터 커서(Cursor)​

UV 에디터에서 사용하는 포인트 형태의 커서로, 특정 위치를 기준으로 툴을 사용하고자 할 때 사용합니다.

커서의 중심점을 클릭드래그하여 이동시킬 수 있습니다.

사용 방법​

  1. UV 엘리먼트를 선택합니다.
  2. Flip Center 툴을 실행합니다.
  3. 선택한 UV 엘리먼트가 중앙 반전됩니다.
참고

커서 활성화되었을 경우​

커서가 활성화 되어 있을 경우, 커서의 위치를 중심으로 중앙 반전됩니다.
