Sew 툴

UV 에디터의 Sew 툴은 선택한 UV 엣지를 공유된 UV 엣지에 붙이는 툴입니다.

참고

공유된(Shared) 엘리먼트​

공유된 엘리먼트는 UV로는 떨어져 있지만, 메시에서는 붙어있거나 인접한 엘리먼트를 의미합니다.

UV 색상

공유된 엘리먼트는 파란색(기본값)으로 표시됩니다.

사용 방법​

  1. 연결할 UV 엣지를 선택합니다.
  2. Sew 툴을 실행합니다.
  3. 선택한 UV 엣지가 공유된 UV 엣지로 이동되며 합쳐집니다.
