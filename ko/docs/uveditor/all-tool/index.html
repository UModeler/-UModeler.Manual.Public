All 툴

UV 에디터의 All 툴은 현재 UV 에디터에 띄워져 있는 모든 UV를 선택하는 툴입니다.

참고

활성화된 엘리먼트 선택에 맞춰 전체 선택이 됩니다.

사용 방법​

버텍스​

  1. UV 에디터에서 버텍스 엘리먼트를 선택합니다.
  2. All 툴을 실행합니다.
  3. 모든 버텍스가 선택되었습니다.

엣지​

  1. UV 에디터에서 엣지 엘리먼트를 선택합니다.
  2. All 툴을 실행합니다.
  3. 모든 엣지가 선택되었습니다.

페이스​

  1. UV 에디터에서 페이스 엘리먼트를 선택합니다.
  2. All 툴을 실행합니다.
  3. 모든 페이스가 선택되었습니다.
팁

영역을 벗어난 UV 확인​

모든 UV들이 전체 선택되기 때문에, Rectangle 툴 버튼을 누른 후 All 툴을 사용하여 편집 영역을 벗어난 UV가 있는지 확인하는 방법이 있습니다.
