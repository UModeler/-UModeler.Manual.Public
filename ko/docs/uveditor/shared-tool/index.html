Shared 툴

UV 에디터의 Shared 툴은 선택된 UV와 공유된 UV 엘리먼트를 선택하는 툴입니다.

참고

공유된(Shared) 엘리먼트​

공유된 엘리먼트는 UV로는 떨어져 있지만, 메시에서는 붙어있거나 인접한 엘리먼트를 의미합니다.

UV 색상

공유된 엘리먼트는 파란색(기본값)으로 표시됩니다.

사용 방법​

버텍스​

  1. 버텍스를 선택합니다.
  2. Shared 툴을 실행합니다.
  3. 선택한 버텍스와 메시를 공유하는 버텍스가 선택되었습니다.

엣지​

  1. 엣지를 선택합니다.
  2. Shared 툴을 실행합니다.
  3. 선택한 엣지와 메시를 공유하는 엣지가 선택되었습니다.
