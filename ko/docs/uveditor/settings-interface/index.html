설정과 인터페이스

1. 툴바​

화면 이동, 트랜스폼 기즈모, UV 엘리먼트 선택 툴, 커서, 피봇 전환이 순서대로 나열되어 있습니다.

화면 이동과 트랜스폼 기즈모​

Hand 툴과 트랜스폼 기즈모로 이루어져 있습니다.

원하는 UV 엘리먼트를 먼저 선택 후 기즈모 중 한 가지를 눌러 나온 기즈모를 조작하여 선택된 엘러먼트에 변형을 줄 수 있습니다.

Hand 툴​

화면을 이동할 때 쓰는 툴입니다. Hand 툴 실행 후 UV 에디터 작업 공간을 클릭드래그 하면 화면을 움직일 수 있습니다. 단축키는 Q

팁

단축키 가운데클릭-드래그​

가운데클릭-드래그 를 이용하면 트랜스폼 기즈모 툴을 사용 중에도 툴 전환 없이 화면 이동이 가능하기 때문에, 가운데클릭-드래그 방식을 권장합니다.

참고

화면 조작 단축키​

  • 화면 이동: Q(Hand 툴)+클릭드래그 , 가운데클릭-드래그
  • 화면 확대축소: 휠회전

Move 툴​

선택한 UV 엘리먼트를 이동시킬 때 사용하는 툴입니다. 단축키는 W

Rotation 툴​

선택한 UV 엘리먼트를 회전시킬 때 사용하는 툴입니다. 단축키는 E

Scale 툴​

선택한 UV 엘리먼트의 크기를 조절할 때 사용하는 툴입니다. 단축키는 R

Rectangle 툴​

선택한 UV 엘리먼트의 크기를 조절할 때 사용하는 툴입니다. 단축키는 T

참고

Scale 툴과 다른 점​

Rectangle 툴은 가상의 사각형에 있는 점들을 조작하여, Scale 툴과는 다른 기준으로 크기 조절이 가능하다는 점이 차이가 있습니다.

UV 엘리먼트 선택​

편집할 UV 엘리먼트를 선택하는 툴들입니다.

트랜스폼 기즈모 툴과 특정 UV 엘리먼트에서 동작하는 툴을 사용하기 위해서는 먼저 UV 엘리먼트를 선택해야합니다.

참고

UV 엘리먼트 선택하기​

UV 엘리먼트를 선택하기 위해서는 원하는 UV 엘리먼트 선택 툴이 활성화된 상태에서, 클릭드래그를 통해 선택 영역을 만들어 선택이 가능합니다.

UV 엘리먼트는 선택 영역 안에 들어오거나, 선택 영역에 걸치는 UV 엘리먼트만 선택이 가능합니다.

UV Vertex 선택​

UV 버텍스를 선택하는 툴입니다.

UV Edge 선택​

UV 엣지를 선택하는 툴입니다.

UV Face 선택​

UV 페이스를 선택하는 툴입니다.

UV Island 선택​

UV 아일랜드를 선택하는 툴입니다.

참고

UV 아일랜드​

UV 아일랜드는 서로 연결되어 있는 UV 엘리먼트를 뜻합니다.

설정과 피벗​

Cursor 툴​

Cursor 툴은 UV 에디터에서 사용하는 포인트 형태의 커서로, 특정 위치를 기준으로 툴을 사용하고자 할 때 사용합니다.

주로 Rotation 툴, Scale 툴 그리고 UV에 관련된 툴 등에서 기즈모의 위치로 쓰입니다. 커서의 중심점을 클릭드래그하여 이동시킬 수 있습니다.

기즈모 전환 툴​

기즈모 위치의 기준을 전환하는 툴로, 누를 때마다 Pivot 속성과 Center 속성이 전환되는 툴입니다.

참고

Cursor 툴이 활성화되어 있을 경우 기즈모 위치가 커서 위치로 고정됩니다.

Pivot

Pivot 속성은 마지막으로 선택된 UV 엘리먼트의 위치를 기즈모의 위치로 사용하는 속성입니다.

팁

특정 UV 엘리먼트의 위치로 기즈모의 위치를 바꾸고 싶다면, 해당 UV 엘리먼트을 선택해제 후 다시 선택하면 됩니다.

Center

Center 속성은 선택된 UV 엘리먼트의 중앙을 기즈모의 위치로 사용하는 속성입니다.

여러 UV 엘리먼트가 선택되었다면 선택된 UV 엘리먼트들의 평균위치가 기즈모 위치로 사용됩니다.

설정​

UV 에디터의 설정 영역을 표시하는 버튼입니다.

2. 설정 영역​

UV 에디터와 관련된 설정을 할 수 있는 영역입니다.

Snap Type​

UV 엘리먼트의 이동, 회전, 크기를 조작할 때 엘리먼트의 위치가 SnapType 속성과 그 외 스냅에 관련된 속성에 따라 스냅 됩니다.

정보

스냅(Snap)​

스냅은 엘리먼트를 정확한 위치에 배치하기 위한 기능입니다. 여러 속성과 미리 지정된 값을 이용하여, 엘리먼트를 특정 위치로 이동하거나, 특정 각도만큼 회전 시키는 등 정확하고 효율적으로 엘리먼트를 편집할 수 있습니다.

  • None
  • World
  • Increment
  • Pixel
속성 탭

SnapType: None​

스냅을 적용하지 않을 때의 속성입니다.

속성 탭

SnapType: World​

UV 에디터 작업 영역의 격자에 맞춰 UV 버텍스가 스냅되는 속성입니다.

GridSnapSize 속성​

UV 엘리먼트 이동, 회전, 크기 조절 시 UV 버텍스가 스냅되는 격자의 간격을 설정하는 속성입니다.

속성 탭

SnapType: Increment​

SnapSize 속성에서 설정된 값만큼 스냅되어 이동, 회전하는 속성으로, 선택된 UV 엘리먼트가 UV 버텍스의 스냅없이 이동, 회전되는 속성입니다.

GridSnapSize 속성​

UV 엘리먼트를 이동시킬 때 적용될 스냅 크기를 설정하는 속성입니다.

RotationSnapSize 속성​

UV 엘리먼트를 회전시킬 때 적용될 스냅 각도를 설정하는 속성입니다.

속성 탭

SnapType: Pixel​

적용된 텍스쳐의 픽셀에 맞춰 UV 버텍스가 스냅되는 속성입니다.

PixelCenter​

텍스쳐의 픽셀 중앙으로 버텍스가 스냅되는 속성입니다.

  • 비활성화할 경우 텍스쳐의 픽셀 가장자리에 스냅 됩니다.
팁

Pixel 속성은 주로 픽셀아트 스타일의 모델을 제작할 때 유용합니다.

주의

Pixel 속성을 쓸 때 스냅 간격은 픽셀아트를 표현한 텍스쳐의 해상도를 따르기 때문에, 표현된 픽셀 크기와 실제 텍스쳐의 해상도가 달라서는 안됩니다.

ShowIslandBoundary​

UV 아일랜드의 바운드 박스를 표시하는 속성입니다.

FillPolygons​

UV 페이스를 반투명으로 표시하는 속성입니다.

팁

UV 페이스를 반투명으로 표시하기 때문에, 반투명 농도 차이로 겹쳐진 UV 페이스를 쉽게 알 수 있습니다.

ShowGrid​

UV 에디터 작업 영역에 격자를 표시하는 속성입니다.

ShowAxes​

UV 에디터 작업 영역의 X,Y축에 해당하는 U축과 V축을 표시하는 속성입니다.

참고

텍스처 공간​

UV축의 시작점부터 큰 간격으로 된 고정된 격자(Fixed Grid Line)가 UV축과 일치하는 지점까지가 텍스쳐 공간을 의미합니다. 대체로 타일링 되는 텍스쳐를 표현하는 게 아니라면, 모든 UV 엘리먼트가 텍스처 공간 안에 배치되어야 합니다.

ShowTexture​

UV 에디터의 Material 속성에서 설정된 머티리얼의 텍스처를 표시하는 속성입니다.

EnableAlphaChannel​

UV 에디터의 Material 속성에서 설정된 머티리얼의 텍스처 알파채널 부분을 제외하여 표시하는 속성입니다.

TexXNum​

UV 에디터의 Material 속성에서 설정된 머티리얼의 텍스처를 좌우 반복으로 표시하는 속성입니다.

TexYNum​

UV 에디터의 Material 속성에서 설정된 머티리얼의 텍스처를 상하 반복으로 표시하는 속성입니다.

Material​

Material 툴로 적용된 머티리얼들을 보여주는 속성으로, 해당 머티리얼이 할당된 UV 엘리먼트만 표시하는 속성입니다.

All​

모든 UV 엘리먼트를 머티리얼 할당과 상관없이 표시하는 속성입니다.
