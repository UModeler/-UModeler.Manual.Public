Painting 2D

Painting 2D 툴는 페인팅 모드와 동기화 되는 에디터 형태의 툴입니다.

실행 방법​

Painting 2D 에디터를 실행하는 방법은 다음과 같습니다.

Source Mesh>Painting2D 툴​

  1. 원본 메시(Source Mesh)를 선택합니다.
  2. 모드 상단 가장 우측에 있는 에디터 버튼을 누릅니다.
  3. 메뉴에서 Painting2D 툴을 실행합니다.
  4. Painting2D 에디터를 확인할 수 있습니다.

오브젝트 선택>열려있는 Painting 2D 에디터 탭 선택​

  1. UModelerX 오브젝트를 선택합니다.
  2. 열려있는 탭들 중 Painting 2D 에디터 탭을 눌러 활성화합니다.
    참고

    UModeler X 오브젝트가 선택되지 않았다면, Painting 2D 에디터가 활성화되지 않습니다.

속성​

ShowGrid​

Painting 2D 작업 영역에 격자를 표시하는 속성입니다.

ShowAxis​

Painting 2D 작업 영역의 X축과 Y축을 표시하는 속성입니다.

ShowWireframe​

현재 선택된 UModeler X 오브젝트의 모든 UV 엘리먼트를 표시하는 속성입니다.

맵소스 선택​

특정 맵소스만 선택하는 속성입니다.
