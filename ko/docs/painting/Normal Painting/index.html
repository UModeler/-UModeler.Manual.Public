Normal Painting

Normal Painting은 매쉬 표면에 입체감을 주기 위해 높이와 높이 정보를 적용하여 디테일을 표현하는 맵소스의 일종입니다.

사용 방법​

  1. Map Source 그룹 속성에서 Add Map Source > Height 버튼을 클릭합니다.

  2. Height Map Source가 추가됩니다. Height as Normal을 체크합니다.

  3. Material Mode를 Preview로 변경하면 실시간으로 Height-Normal이 적용되는것을 확인할 수 있습니다.

Properties 그룹 속성​

Texture​

브러시에 텍스처를 적용하여 커스터마이징할 수 있는 기능입니다.

Value​

Brush의 높이값을 결정합니다. Background Value 0.5값을 기준으로 0에 가까울수록 음각으로, 1에 가까울수록 양각으로 표현됩니다.

Height Map Source 그룹 속성​

Background Value​

Background의 높이값을 결정합니다.

Height as Normal Strength (Height as Normal)​

높이맵 정보로 노말맵을 생성할 때의 경사도 강도를 조절합니다.
