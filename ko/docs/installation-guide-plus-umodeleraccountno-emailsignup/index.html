Plus 설치 : E-mail 이용

노트

다른 방식의 UModeler X Plus 설치 방법은 아래 링크의 페이지를 참고하시기 바랍니다.

  • 구글 계정으로 가입 후 설치하기
  • 기존 UModeler X Beta/Basic 사용자

UModeler X Plus 설치하기​

1. 유니티 에셋 스토어에서 UModeler X Basic 버전을 내 에셋에 추가합니다.​

2. 유니티 프로젝트를 엽니다.​

3. 상단 메뉴의 window > Package Manager를 연 뒤 UModeler X Plus 패키지로부터 UModeler Hub를 다운로드 하고 임포트 합니다.​

4. 다운로드 후 Scene 상단 중앙에 생성되는 UModeler Hub 아이콘을 클릭하여 윈도우를 엽니다.​

5. 하단의 Create a New Account를 클릭합니다.​

6. 이메일 및 비밀 번호 등의 필요한 정보 입력, 약관에 동의한 후 Create a UModeler ID 버튼을 클릭합니다.​

7. 인증 메일이 발송됩니다. 기재한 이메일로 로그인 한 뒤 인증 메일을 확인하여 인증절차를 완료하면 다음 페이지로 이동합니다.​

8. Unlock your UModeler X Plus 페이지에서 구매한 Plus 버전의 Order ID 혹인 Invoice 번호를 입력한 후 Unlock 버튼을 클릭합니다.​

[Invoice 번호 확인하는 방법]

9. UModeler X Plus의 Install버튼이 활성화 됩니다. 클릭하여 설치를 진행합니다.​

경품 에셋 다운로드 가이드

UModeler X Plus 사용자는 허브에서 유료 에셋을 무료로 다운로드 받을 수 있습니다.
