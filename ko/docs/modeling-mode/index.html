모델링 모드

모델링(Modeling) 모드는 에디터블 메시의 기본이 되는 모드로, 메시 편집이 주된 기능입니다. 이 모드는 여러 기능을 하는 툴과 툴의 세부 설정을 뜻하는 속성, 그리고 툴들을 묶은 그룹으로 분류됩니다.

참고

엘리먼트(Element)​

툴을 사용하기 위해서는 엘리먼트(Element)라고 불리는 3D 기본 요소를 선택하거나 이용해야 합니다.

엘리먼트는 버텍스(Vertex), 엣지(Edge), 페이스(Face), 오브젝트(Object)로 구성되어 있습니다.

버텍스는 3D 오브젝트의 꼭지점을 의미하며, 엣지는 두 버텍스로 이루어진 모서리를 나타냅니다. 페이스는 세 개 이상의 엣지를 연결하여 채워진 면을 뜻합니다.

UModelerX는 버텍스, 엣지, 페이스를 편집하고 생성하기 위한 여러 가지 툴을 제공합니다.

각 엘리먼트를 이동, 회전, 크기 변경하는 것은 Unity 에디터의 조작과 동일합니다.

모델링 모드의 그룹​

Element 그룹​

Element 그룹은 모델링 모드 최상단에 위치하며, 각 엘리먼트(Vertex, Edge, Face, Object)에 관한 속성을 모아 놓은 그룹입니다.

각 엘리먼트에 따라 보여지는 속성이 서로 다릅니다.

Primitive Shapes 그룹​

Primitive Shapes 그룹은 기본 도형 메시를 생성할 수 있는 툴을 모아 놓은 그룹입니다.

자세한 내용은 Primitive Shapes 그룹 페이지를 참조하세요.

Drawing 그룹​

Drawing 그룹은 평면 형태의 도형을 생성할 수 있는 툴을 모아 놓은 그룹입니다.

자세한 내용은 Drawing 그룹 페이지를 참조하세요.

Selection 그룹​

Selection 그룹은 각 엘리먼트(Vertex, Edge, Face, Object)의 선택에 관련된 툴을 모아놓은 그룹입니다.

자세한 내용은 Selection 그룹 페이지를 참조하세요.

Add 그룹​

Add 그룹은 실행하면 각 엘리먼트(Vertex, Edge, Face)가 추가되는 툴들을 모아놓은 그룹입니다.

주로 선택된 엘리먼트가 추가되거나, 툴에 정의된 엘리먼트가 추가되는 툴들이 포함되어 있습니다.

자세한 내용은 Add 그룹 페이지를 참조하세요.

Remove 그룹​

Remove 그룹은 실행하면 각 엘리먼트(Vertex, Edge, Face)가 삭제되는 툴들을 모아 놓은 그룹입니다.

주로 선택된 엘리먼트가 삭제되거나, 툴에 의해 엘리먼트가 합쳐지는 툴들이 포함되어 있습니다.

자세한 내용은 Remove 그룹 페이지를 참조하세요.

Deform 그룹​

Deform 그룹은 표면에 변형을 가하는 브러시 형태의 툴을 모아 놓은 그룹입니다.

자세한 내용은 Deform 그룹 페이지를 참조하세요.

Surface 그룹​

Surface 그룹은 표면에 대한 툴을 모아 놓은 그룹입니다.

주로 버텍스 컬러, 머티리얼 설정, UV 에디터와 함께 쓸 수 있는 툴들이 포함되어 있습니다.

자세한 내용은 Surface 그룹 페이지를 참조하세요.

Misc 그룹​

Misc 그룹은 다른 그룹에 속하지 않는 기능의 툴을 모아 놓은 그룹입니다.

자세한 내용은 Misc 그룹 페이지를 참조하세요.

Multiple Object 그룹​

Multiple Object 그룹은 다수의 UModeler X 오브젝트가 선택된 상태에서 사용할 수 있는 툴을 모아 놓은 그룹입니다.

자세한 내용은 Multiple Object 그룹 페이지를 참조하세요.
