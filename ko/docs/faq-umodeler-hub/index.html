유모델러 허브

회원 가입이 안됩니다.​

베타 테스터 프로모션 신청을 하셨나요?​

현재는 베타 테스트를 신청한 사용자만 회원 가입을 할 수 있습니다. UTC 2023/03/30 09:00:00 시간까지 신청한 사용자에게 베타 테스트 서비스를 제공합니다.

정상적인 이메일인가요?​

우리 서비스가 발송하는 이메일을 수신한 사용자만 서비스를 이용할 수 있습니다.

이메일을 수신할 수 없는 이유는 다양합니다. 일반적으로 베타 테스터 이메일 신청 시 오탈자로 인해 베타 테스터 이메일을 받지 못할 수 있습니다. 간혹 이메일 서비스 제공자에 의해 우리가 발송한 이메일의 수신이 차단될 수 있습니다.

베타 테스터 신청 기간을 확인하세요.​

우리 서비스는 베타 테스터를 신청한 사용자에게 일시적으로 서비스를 제공합니다. UTC 2023/03/30 09:00:00 시간까지 신청한 사용자에게 베타 테스트 서비스를 제공합니다.

스팸함에 이메일이 도착했나요?​

우리 서비스가 발송한 이메일이 스팸함에 있는지 확인해 보세요. 우리 서비스는 사용자가 이메일을 제대로 수신할 수 없는 상태인 경우 더이상 이메일을 발송하지 않습니다. 만약 이메일이 스팸함에 도착했다면 더 이상 스팸으로 분류되지 않도록 조치를 취해 주세요.

로그인이 안됩니다.​

인터넷 접속이 원활한지 확인 하십시오.​

로그인 기능은 인터넷 접속이 가능한 환경에서 로그인이 가능합니다.

기업 내 인터넷 환경 또는 HTTP Proxy 환경인지 확인 하십시오.​

일부 기업 내 인터넷 환경인 경우 우리 서비스에 접근하기 위해 네트워크 관리자에게 https://api.umodeler.com도메인 접속을 할 수 있도록 요청하세요. 다만, HTTP Proxy 환경인 경우 우리 서비스는 아직 HTTP Proxy 설정을 지원하지 않습니다. 빠른 시일내에 HTTP Proxy 설정 기능을 지원할 예정입니다.

패키지 다운로드가 안됩니다.​

인증된 사용자인가요?​

패키지 다운로드는 인증된 사용자에게만 제공됩니다. 로그인 가능한 계정으로만 패키지를 다운로드 할 수 있습니다.

계속 문제가 발생한다면 인증이 만료되었을 수 있습니다. 로그아웃 후 다시 로그인하여 확인하십시오.

패키지 다운로드가 너무 오래 걸립니다.​

제공받는 인터넷 서비스의 품질 확인이 필요합니다.​

우리 서비스는 전세계 모든 국가에서 안정적인 패키지 다운로드 속도를 제공하기 위해 CDN 으로 지리적으로 가장 원활한 서버로부터 패키지를 다운로드 합니다.

일반적으로 사용자의 인터넷 대역폭 또는 속도의 제한으로 다운로드가 오래 걸릴 수 있습니다.

이 경우 인터넷 서비스 사업자에게 문의 하십시오.

서비스 접속이 안되는 것 같습니다.​

기업 인터넷 환경이나 HTTP Proxy 설정이 필요한가요?​

일부 기업 내 인터넷 환경인 경우 우리 서비스에 접근하기 위해 네트워크 관리자에게 문의하세요.

우리 서비스는 아직 HTTP Proxy 설정을 지원하지 않습니다.

빠른 시일내에 HTTP Proxy 를 지원할 예정입니다.

그래도 문제가 계속 발생합니다.​

언제든지 이메일로 문의 주세요.​

언제든지 이메일 혹은 디스코드 이슈 포럼을 통해 문의 주세요.

발생하는 문제에 대해 조치할 수 있도록 contact.support@umodeler.com 이메일 문의를 주시거나 UModeler X 디스코드 서버 이슈 포럼(링크)에 문의 주세요.

한국 시간으로 워킹 데이 기준 최대 7일 이내에 문제를 해결할 수 있도록 도와 드리겠습니다.
