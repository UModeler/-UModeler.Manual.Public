Mirror 모디파이어

Mirror 모디파이어는 축을 기준으로 대칭된 메시를 복제하는 모디파이어입니다.

사용 방법​

  1. 모디파이어 추가 버튼을 누릅니다.
  2. Mirror 모디파이어를 선택합니다.
  3. Mirror Axis 속성에서 원하는 축을 선택합니다.
  4. 선택한 축에 맞춰 대칭된 메시가 복제됩니다.

속성​

Mirror Axis​

X​

X 축을 대칭 기준으로 설정하는 속성입니다.

Y​

Y 축을 대칭 기준으로 설정하는 속성입니다.

Z​

Z 축을 대칭 기준으로 설정하는 속성입니다.

Flip​

복제하는 축 방향을 바꾸는 속성입니다.

Display Overlay In Source Mesh​

원본 메시에서 선택된 Mirror 모디파이어의 축을 확인할 수 있는 속성입니다.

Angle along Y​

Y축 기준으로 축을 회전하는 속성입니다.

Distance​

피벗부터 축까지의 거리에 대한 속성입니다.

참고

축에 표시된 기즈모를 움직여 Distance 속성을 변경할 수도 있습니다.

사용 예시​

이전
Shell 모디파이어
다음
FFD 모디파이어
