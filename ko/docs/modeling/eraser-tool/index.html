Eraser 툴

개요

Eraser 툴은 선택된 엘러먼트를 지울 때 사용하는 툴입니다

엘리먼트 선택 후 Delete 를 눌러 지울 수 있습니다.

사용 방법

버텍스​

  1. 지우고 싶은 버텍스를 선택합니다
  2. Eraser 툴을 실행합니다
  3. 선택한 버텍스가 지워집니다

엣지​

  1. 지우고 싶은 엣지를 선택합니다
  2. Eraser 툴을 실행합니다
  3. 선택한 엣지가 지워집니다

페이스​

  1. 지우고 싶은 페이스를 선택합니다
  2. Eraser 툴을 실행합니다
  3. 선택한 페이스가 지워집니다
