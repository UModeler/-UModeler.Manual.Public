Slice Cut 툴

Slice Cut 툴은 가상의 평면을 생성하여 그와 교차하는 엘리먼트에 엣지를 생성하는 툴입니다.

사용 방법​

  1. Slice Cut 툴을 실행합니다.
  2. 활성화된 반투명 평면을 원하는 엣지가 생성될 수 있게 기즈모를 움직이거나 회전합니다.
  3. Cut 버튼을 눌러 엣지를 생성해 줍니다.

속성​

Offset​

가상 평면의 위치를 설정하는 속성입니다.

Rotation​

가상 평면의 회전을 설정하는 속성입니다.

Display​

메시의 표시 방식을 선택할 수 있는 속성입니다.

XRay Mode​

페이스를 반투명으로 표시합니다.

Reset Plane​

가상 평면의 위치와 회전을 초기화하는 속성입니다.

Cut​

현재의 가상 평면 상태로 Slice Cut 툴을 실행하는 속성입니다.

새로운 기능​

버전 0.18.0부터 View Cut, Action After Cut 그리고 각도 스냅(Angle Snap) 기능이 새롭게 추가되었습니다.

View Cut​

마우스 드래그로 카메라 방향과 일치하는 가상 평면을 만드는 기능입니다.

Action After Cut​

Cut을 한 이후의 동작을 정의합니다.

  • Split Only : 가상 평면을 기준으로 절단합니다.
  • Remove Above : 가상 평면 윗 부분을 제거하고 절단면을 채웁니다.
  • Remove Below : 가상 평면 아랫 부분을 제거하고 절단면을 채웁니다.

Angle Snap​

Ctrl키를 누른 상태로 마우스를 드래스 하면 유니티 에디터의 Incremental Snapping의 회전(Rotate) 값에 맞춰서 가상 평면이 바라보는 시점 기준으로 스냅됩니다.
