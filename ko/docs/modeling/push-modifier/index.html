Push 모디파이어

Push 모디파이어는 페이스를 노멀방향 또는 반대방향으로 이격시킬때 사용하는 모디파이어 입니다.

사용 방법​

  1. 모디파이어 추가 버튼을 누릅니다.
  2. Push 모디파이어를 선택합니다.
  3. Push Size 속성을 변경합니다.
  4. 페이스가 이격 되어 커지거나 작아집니다.

속성​

Push Size​

페이스의 이격 거리에 대한 속성입니다. 0일 경우 변화가 없습니다.

사용 예시​

이전
Mirror 모디파이어
다음
Shell 모디파이어
