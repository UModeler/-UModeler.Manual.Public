Detach 툴

Detach 툴은 선택된 페이스만 메시에서 떼어내거나, 현재 편집 중인 오브젝트에서 떼어내어 새로운 오브젝트로 분리할 때 사용하는 툴입니다.

사용 방법​

Detach 툴은 설정된 속성에 따라 바로 적용되는 툴로, 실행하기 전에 속성을 변경하여 사용해야 합니다. 아래 속성에 따른 사용 방법을 참고하세요.

팁

변경한 속성은 다시 변경할 때까지 그대로 유지되기 때문에, 자주 사용하는 속성으로 변경해두면 사용하기 편리합니다.

속성​

참고

Detach 툴 아이콘을 Alt+클릭하면 나오는 속성창에서 속성을 변경할 수 있습니다.

Detach Type: To Elements​

연결된 메시에서 분리​

  1. 페이스를 선택합니다.
  2. Deatch 툴을 실행합니다.
  3. 선택한 페이스가 연결된 메시에서 분리됩니다.

Detach Type: To Object​

분리와 함께 오브젝트 생성​

  1. 페이스를 선택합니다.
  2. Deatch 툴을 실행합니다.
  3. 페이스가 분리되면서 선택한 페이스로 새로운 오브젝트가 생성됩니다.
