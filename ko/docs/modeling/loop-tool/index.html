Loop 툴

Loop 툴은 선택한 엘리먼트와 정렬된 방향의 엘리먼트를 연결하여 선택할 수 있는 툴입니다.

사용 방법​

엣지​

  1. 한 개 이상의 엣지를 선택합니다.

  2. Loop 툴을 실행합니다.

    참고

    엣지의 Loop 툴 단축키

    엣지를 더블클릭 합니다.

  3. 선택한 엣지와 연결된 엣지가 선택됩니다.

페이스​

  1. 원하는 방향에 맞춰 2개 이상의 인접한 페이스를 선택합니다.

  2. Loop 툴을 실행합니다.

    참고

    페이스의 Loop 툴 단축키

    페이스를 선택 후 원하는 방향의 페이스를 Shift+더블클릭 합니다.

  3. 선택된 페이스의 진행 방향에 있는 페이스가 선택됩니다.

속성​

참고

Loop 툴 아이콘을 Alt+클릭하면 나오는 속성창에서 속성을 변경할 수 있습니다.

Select Type​

At One Go​

At One Go 속성은 Loop 툴 선택 조건에 따라 모든 엣지와 페이스를 한 번에 선택할 수 있는 속성입니다.

엣지

페이스

Grow​

Grow 속성은 Loop 툴 아이콘을 클릭할 때마다 범위를 늘리는 방식으로 선택하는 속성입니다.

참고

페이스는 Grow 속성이 적용되지 않습니다.

사용 예시​

엣지​

Select Type: At One Go​

Select Type: Grow​

페이스​

Select Type: At One Go​
