Hotspot Texturing 툴

Hotspot Texturing 툴은 지정된 텍스처를 활용하여 절차적으로 빠르고 효율적으로 레벨을 제작할 수 있는 도구입니다.

사용 방법​

  1. 사용하고자 하는 모듈화 된 텍스처를 Project Panel에서 메시로 드래그하여 적용합니다.

  1. Project Panel에서 우클릭하여 Hotspot Layout을 만들어줍니다.

  1. Hotspot Layout 할당

3-1. Hotspot 툴을 실행합니다.

3-2. Hotspot Object에 생성한 Hotspot Layout을 할당합니다.

3-3. 해당 영역을 클릭하거나 Editor의 Hotspot Editor를 클릭하여 Hotspot Editor를 활성화 합니다.

  1. HotspotLayout Editor 우측상단의 톱니바퀴를 누르고 Temporary Texture에 사용할 텍스처를 할당해주세요.

노트

Snap Type & Grid Snap Size​

Snap Type: 작업대상이 어떤 기준에 맞춰 Snap 될지 설정합니다.

Grid Snap Size: 픽셀 단위로 선택할 수 있는 영역의 크기를 정의하며, 텍스처에 맞는 정확한 영역을 지정할 수 있도록 도와줍니다.

하단의 매뉴얼은 Snap Type을 UV로 설정하고, Grid Snap Size를 1/256 크기로 설정하여 진행합니다.

노트

Layout 그룹​

New Layout: 사용할 영역을 나눌 수 있습니다.

Edit Layout: 지정한 영역을 수정할 수 있습니다.

Remove Layout: 잘못 지정한 영역을 삭제할 수 있습니다.

노트

DefaultMag란?​

DefaultMag는 텍스처의 크기와 3D 월드 상의 크기를 매칭하는 배율 값입니다.

  • DefaultMag = 1: UV 공간의 1x1 크기가 3D 월드의 1x1 유닛에 대응됩니다.
  • DefaultMag > 1: 텍스처가 확대되어 적용됩니다.
  • DefaultMag < 1: 텍스처가 축소되어 더 촘촘히 매핑됩니다.
팁

설정 팁:

  • 기본 권장 값: 1~5.
  • 값이 너무 크면 텍스처가 뭉개져 보이고, 너무 작으면 흐릿하게 보일 수 있습니다.
  • 테스트를 통해 적절한 값을 찾아주세요.
  • 텍스처 품질과 퍼포먼스를 고려해 설정하세요!

  1. New Layout 기능을 사용하여 텍스처가 메시의 스케일 변화에 맞춰 적합한 영역을 자동으로 배치하도록 설정합니다.

핫스팟 텍스처링을 사용하는 세 가지 방법​

A: Auto Hotspot in Build​

  1. UV Editor 윈도우를 활성화 합니다.

  2. Unwrap 패널에서 UV가 없는 상태로 만들어야 합니다. 만약 UV가 이미 존재한다면 UV를 전체 선택한 뒤 Cancel 아이콘을 눌러 제거합니다.

팁

Tip: Preferences의 Editable Mesh에서 Auto Unwrap Shape 옵션을 비활성화하면, 새로 생성되는 도형에 자동 언랩 기능이 적용되지 않도록 설정할 수 있습니다.

  1. 매시를 선택하고 Surface > Hotspot > Auto Hotspot in Build를 클릭합니다.

  1. Face를 이동시켜보면, 할당한 텍스처를 기반으로 절차적으로 변경되는 것을 확인할 수 있습니다.

B: Auto Hotspot Selected Faces​

  1. 매시를 선택하고 Surface > Hotspot > Auto Hotspot Selected Face를 클릭합니다.

  2. 메시의 페이스를 다시 움직이면 텍스처가 늘어나거나 찌그러질 수 있으므로, 메시를 변형할 때마다 Auto Hotspot Selected Faces를 다시 눌러야 합니다. 이 기능은 선택된 페이스들에만 적용됩니다.

C: Fit Selected Hotspot Faces​

  1. HotspotLayout Editor 윈도우에서 Edit Layout을 선택한 뒤, 하나의 Face를 클릭하면 Tool Properties 창에서 Layout Name을 설정할 수 있습니다. 해당 매뉴얼에서는 이 이름을 "test"로 지정했습니다.

  1. Inspecter의 UModeler X Component에서 Surface > Hotspot을 클릭합니다.

  2. Tool Properties의 Selected Layout을 클릭하여 지정한 레이아웃을 클릭합니다.

  1. Fit Selected Hotspot Face를 클릭합니다.

  1. 뷰포트에서 선택했던 Face가 그대로 표시되는 것을 확인할 수 있습니다.

정리​

  • A. Auto Hotspot in Build: 메시의 Face가 움직일 때 절차적으로 텍스처 영역이 실시간으로 자동 업데이트되는 방식입니다.
  • B. Auto Hotspot Selected Faces: 선택한 Face를 기준으로 텍스처 영역을 수동으로 변경하는 방식입니다.
  • C. Fit Selected Hotspot Faces: Layout을 지정하여 텍스처를 정확하고 의도적으로 조정하는 방식입니다.
