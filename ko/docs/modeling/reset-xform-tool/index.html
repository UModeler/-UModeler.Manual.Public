Reset X Form 툴

Reset X Form 툴은 선택된 오브젝트의 트랜스폼인 Position, Rotation, Scale 값을 초기화하는 툴입니다.

사용 방법​

  1. Reset X Form 툴을 실행합니다.
  2. 원하는 속성 버튼을 클릭합니다.
  3. Transform 컴포넌트의 값이 선택한 속성에 따라 초기화된 것을 볼 수 있습니다.

속성​

Reset All​

오브젝트의 Position, Rotation, Scale 값을 전부 초기화하는 속성입니다.

Reset Position​

오브젝트의 Position 값을 초기화하는 속성입니다.

Reset Rotation​

오브젝트의 Rotation 값을 초기화하는 속성입니다.

Reset Scale​

오브젝트의 Scale 값을 초기화하는 속성입니다.
