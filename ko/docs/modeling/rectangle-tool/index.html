Rectangle 툴

Rectangle 툴은 사각 평면 메시를 생성하는 툴입니다.

사용 방법​

Rectangle 툴은 두 가지 방법으로 사각 평면을 생성할 수 있습니다.

  1. 클릭드래그를 통한 생성, 편집
  2. One Click Build를 이용한 생성, 편집

클릭드래그를 통한 생성, 편집​

  1. Rectangle 툴을 선택합니다.
  2. 씬에 마우스로 클릭드래그해 원하는 크기의 평면을 생성합니다.
  3. 원하는 크기로 평면이 생성되었다면 클릭을 마쳐 메시 생성을 마칩니다.
  4. Rectangle 툴의 속성들을 조절한 후 엘리먼트를 선택하거나, 다른 툴을 눌러 마무리합니다.
    참고

    엘리먼트를 선택하거나 다른 툴을 눌러 마무리하기 전까지 Rectangle 툴의 속성을 이용해 세부조절을 계속할 수 있습니다.

One Click Build와 속성을 통한 생성, 편집​

  1. Rectangle 툴을 선택합니다.
  2. Rectangle 툴의 속성에 원하는 값을 입력합니다.
  3. One Click Build 버튼을 눌러 씬에 도형을 추가합니다.
  4. 추가로 Rectangle 툴의 속성들을 조절한 후 엘리먼트를 선택하거나, 다른 툴을 눌러 마무리합니다.
참고

엘리먼트를 선택하거나 다른 툴을 눌러 마무리하기 전까지 Rectangle 툴의 속성을 이용해 세부 조절을 계속할 수 있습니다.

속성​

Width​

생성되는 도형의 가로 크기를 설정하는 속성입니다.

오브젝트의 기준으로 Z축에 해당하는 방향으로, 단위는 미터입니다.

Depth​

생성되는 도형의 세로 크기를 설정하는 속성입니다.

오브젝트의 기준으로 X축에 해당하는 방향으로, 단위는 미터입니다.

Width Segs​

생성되는 도형의 가로로 분할되는 조각 수를 설정하는 속성입니다.

Depth Segs​

생성되는 도형의 세로로 분할되는 조각 수를 설정하는 속성입니다.
