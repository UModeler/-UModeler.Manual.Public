Vertex Color 툴

Vertex Color 툴은 선택한 버텍스에 버텍스 컬러를 적용하는 툴입니다.

참고

버텍스 컬러(Vertex Color)​

버텍스 컬러는 버텍스에 저장되는 색상 정보입니다. 버텍스 간의 색상이 자동으로 보간되며, 다양한 용도로 활용될 수 있습니다.

사용 방법​

  1. Vertex Color 툴을 실행합니다.
  2. View Vertex Color 속성을 활성화하면, 오브젝트의 버텍스 컬러가 표시됩니다.
  3. 버텍스 컬러를 적용할 버텍스를 선택합니다.
  4. Vertex Color 속성을 클릭하여 원하는 색상으로 변경합니다.
  5. Apply Color 버튼을 클릭하여, 버텍스 컬러를 적용합니다.

속성​

VertexColor​

View Vertex Color​

오브젝트의 버텍스 컬러를 표시하는 속성입니다.

Vertex Color​

버텍스 컬러를 지정할 수 있는 속성입니다.

Select​

버텍스의 선택 방식을 설정할 수 있는 속성입니다.

Backface Select​

반대편을 향하는 버텍스들의 선택 여부를 설정할 수 있습니다.

Backface Select 속성이 꺼져 있을 때는 반대편을 향하는 버텍스들이 선택되지 않습니다.

Backface Select 속성이 켜져 있을 때는 반대편을 향하는 버텍스들도 선택됩니다.

Camera Based Select​

카메라 기준으로 보여지는 버텍스들만 선택하는 속성입니다.

Camera Based Select 속성 꺼져 있을 때는 카메라에 가려진 버텍스들도 선택됩니다.

Camera Based Select 속성이 켜져 있을 때는 카메라에 가려진 버텍스들은 선택되지 않습니다.
