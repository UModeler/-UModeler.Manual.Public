Subdivide 모디파이어

Subdivide 모디파이어는 현재 페이스를 4등분하는 동시에 굴곡을 부드럽게 변형하는 모디파이어입니다.

참고

페이스 Element 그룹의 Smoothing Groups 속성처럼 페이스가 연결된 것처럼 보이게 하는 것이 아닌, 실제 페이스의 개수가 증가합니다.

사용 방법​

  1. 모디파이어 추가 버튼을 누릅니다.

  2. Subdivide 모디파이어를 선택합니다.

  3. Subdivision Level 속성에서 원하는 단계의 버튼을 선택합니다.

    주의

    전체 페이스를 단계당 4등분하므로, Subdivision Level 속성의 단계에 따라 많은 페이스로 인한 3D 오브젝트의 최적화 문제, 순간적인 에디터 속도 저하 등이 나타날 수 있으므로, 낮은 단계부터 실행하는것을 권장합니다.

  4. 페이스가 추가되며 굴곡이 부드러워 졌습니다.

속성​

Subdivision Level​

Subdivide 단계에 대한 속성입니다.

각 단계마다 페이스의 개수는 4배로 증가하며 굴곡이 부드러워집니다.

Interpolation​

None​

Edge Only​

모서리 부분과 엣지 부분을 구분하지 않고 굴곡이 정해지는 속성입니다.

결과적으로 모든 꼭지점이 부드럽게 표현됩니다.

Edge And Corner​

꼭지점 부분과 엣지 부분을 구분하여 굴곡이 정해지는 속성입니다.

UV Interpolation​

펼쳐진 UV에 대한 굴곡 방식을 설정하는 속성입니다.

참고

UV가 펼쳐져 있지 않다면 작동하지 않습니다.

정보

UV가 펼쳐져 있지 않더라도 오브젝트의 공간 좌표, 노멀 방향을 기반으로 자동으로 생성된 UV 텍스처 좌표를 가지게 됩니다.

아래 이미지는 각 속성에 따른 UV의 변화 예시입니다.

Auto Smoothing Group​

Subdivision Level에 의해 추가된 페이스에 대해, 자동으로 스무딩 그룹을 적용하여 표면이 부드럽게 보이게 하는 속성입니다.

팁

Subdivision Level의 값을 올리게 되면 그만큼 폴리곤이 많아지기 때문에, Auto Smoothing Group 속성을 활성화 상태에서 표현 수준을 보며 Subdivision Level의 값을 올리는 것이 최적화에 좋습니다.
