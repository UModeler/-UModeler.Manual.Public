Room 툴

Room 툴은 방 형태의 모델을 생성하는 툴입니다.

사용 방법​

Primitive Shapes 그룹의 기본 도형 생성 툴들은 두 가지 방법으로 도형을 생성할 수 있습니다.

  1. 클릭드래그를 통한 생성, 편집
  2. One Click Build를 이용한 생성, 편집

클릭드래그를 통한 생성, 편집​

  1. Room 툴을 선택합니다.
  2. 씬에서 원하는 위치에 마우스로 클릭드래그하여 원하는 크기의 밑면을 생성합니다.
  3. 원하는 크기로 밑면이 생성되었다면 클릭을 떼고 마우스를 위로 움직여 원하는 높이로 메시를 생성합니다.
  4. 원하는 높이가 되면 다시 클릭하여 메시 생성을 마칩니다.
  5. Room 툴의 속성 중 Thickness을 조절하여 벽의 두께를 조절합니다.
    • 이때, XRay Mode를 활성화하면 페이스가 반투명으로 표시되어 조절하기 더 쉬워집니다.
  6. Box 툴의 나머지 속성들을 조절한 후 엘리먼트를 선택하거나, 다른 툴을 눌러 마무리합니다.
참고

엘리먼트를 선택하거나 다른 툴을 눌러 마무리하기 전까지 Room 툴의 속성을 이용해 세부조절을 계속할 수 있습니다.

One Click Build와 속성을 통한 생성, 편집​

  1. Room 툴을 선택합니다.
  2. Room 툴의 속성에 원하는 값을 입력합니다.
  3. One Click Build 버튼을 눌러 씬에 도형을 추가합니다.
  4. 추가로 Room 툴의 속성들을 조절한 후 엘리먼트를 선택하거나, 다른 툴을 눌러 마무리합니다.
참고

엘리먼트를 선택하거나 다른 툴을 눌러 마무리하기 전까지 Room 툴의 속성을 이용해 세부조절을 계속할 수 있습니다.

속성​

Width​

생성되는 방 도형의 가로 크기를 설정하는 속성입니다.

오브젝트의 기준으로 X축에 해당하는 방향으로, 단위는 미터입니다.

Height​

생성되는 방 도형의 높이를 설정하는 속성입니다.

오브젝트의 기준으로 Y축에 해당하는 방향으로, 단위는 미터입니다.

Depth​

생성되는 방 도형의 세로 크기를 설정하는 속성입니다.

오브젝트의 기준으로 Z축에 해당하는 방향으로, 단위는 미터입니다.

Thickness​

생성되는 방 도형의 벽의 두께를 설정하는 속성입니다.

Display​

메시의 표시 방식을 선택할 수 있는 속성입니다.

XRay Mode​

페이스를 반투명으로 표시합니다.
