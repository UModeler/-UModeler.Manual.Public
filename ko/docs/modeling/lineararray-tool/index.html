Linear Array 툴

Linear Array 툴은 선택한 엘리먼트(Vertex, Edge, Face)를 복사하여 선형 방향으로 배열하는 툴입니다.

사용 방법​

  1. 복사할 엘리먼트를 선택합니다.
  2. Linear Array 툴을 실행합니다.
  3. Array Type 를 선택합니다.
  4. Length Type 를 선택합니다.
  5. 적절한 수치로 조절합니다.

속성​

Array Type​

Linear Array 툴에 영향을 받을 엘리먼트 종류를 설정하는 속성입니다

Array Type: To Elements​

현재 메시 안에 새 엘리먼트 배열 생성​

  1. 페이스를 선택합니다.
  2. Linear Array 툴을 실행합니다.
  3. 선택한 엘리먼트가 연결된 메시에서 새 엘리먼트 배열을 만듭니다.

Array Type: To Object​

현재 메시에서 독립된 오브젝트 생성​

  1. 페이스를 선택합니다.
  2. Linear Array 툴을 실행합니다.
  3. 현재 메시에서 독립적된 새로운 오브젝트 배열이 생성됩니다.

Length Type​

새로 생성될 배열의 가장 끝 객체의 위치를 결정하는 인자입니다. Total, Interval 두 가지 타입이 있습니다.

  • Total : Total 값만큼 떨어진 곳이 끝 객체의 위치
  • Interval : angle X batch count 값만큼 이동한 거리/ 회전한 각도가 마지막 객체가 위치할 곳

Direction​

원본 엘리먼트의 Pivot을 원점으로한 3차원 방향 벡터의 좌표(성분)로 이 벡터를 따라 Array 됩니다.

Length​

엘리먼트 간의 간격입니다.

Batch Count​

최종적으로 생성될 엘리먼트 또는 오브젝트의 수를 설정합니다 (2 ~ 1000).

Select​

페이스의 선택 방식을 설정할 수 있는 속성입니다.

Backface Select​

반대편을 향하는 페이스들의 선택 여부를 설정할 수 있습니다.

Backface Select 속성이 꺼져 있을 때는 반대편을 향하는 페이스들이 선택되지 않습니다.

Backface Select 속성이 켜져 있을 때는 반대편을 향하는 페이스들도 선택됩니다.

Camera Based Select​

카메라 기준으로 보여지는 페이스들만 선택하는 속성입니다.

Camera Based Select 속성이 꺼져 있을 때는 카메라에 가려진 페이스들도 선택됩니다.

Camera Based Select 속성이 켜져 있을 때는 카메라에 가려진 페이스들은 선택되지 않습니다.
