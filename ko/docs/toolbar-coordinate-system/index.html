좌표 시스템

UModelerX는 유니티의 좌표 시스템을 포함한 총 5개의 좌표 시스템이 있습니다.

Normal, Local, Global, Screen, 그리고 Custom. 이들은 각각의 이름과 아이콘으로 표시됩니다.

UModelerX 툴바의 세 번째 버튼을 클릭하면 팝업이 등장합니다.

UModelerX 좌표 시스템​

  • Normal : 선택한 엘리멘트의 수직한 좌표 시스템입니다. 여러 개의 엘리멘트를 선택하면 모든 수직 방향의 평균을 나타냅니다.

  • Local : 유니티의 Local과 동일한 좌표 시스템입니다. Transform의 회전에 따라 방향이 변합니다.

  • Global : 유니티의 Global과 동일한 좌표 시스템입니다. 항상 x, y, z축의 방향을 나타냅니다.

  • Screen : 스크린 방향을 나타내는 좌표 시스템입니다. 카메라를 돌려도 항상 같은 방향을 나타냅니다.

  • Custom : 사용자 좌표 시스템입니다. 항상 Custom Coordinate Tool을 통해 설정한 좌표 시스템을 나타냅니다. 설정 하지 않았다면, Global 좌표 시스템과 동일한 설정입니다.

노트

Global과 Local 좌표 시스템은 Unity의 좌표 시스템과 연동됩니다.
