페인팅 모드

페인팅(Painting)모드는 모델에 색상을 칠하는 모드로, 레이어와 브러시 등을 이용하여 색상을 표현하고 텍스처를 생성할 수 있습니다.

페인팅 모드 시작​

페인팅 모드를 사용하기 위해서는 먼저 설정 단계를 진행해야 합니다.

UV Unwrap​

모델에 UV가 펼쳐져 있지 않을 경우 표시되는 속성입니다. UV 에디터로 UV가 편집된 상태라면 표시되지 않습니다.

Materials​

페인팅을 진행할 머티리얼을 선택하는 속성입니다.

Startup Resolution​

페인팅을 진행할 때의 텍스처 해상도를 선택하는 속성입니다. 페인팅 도중 변경 가능합니다.

Startup Material Template​

페인팅에 사용될 미리 정의된 맵소스를 설정하는 속성입니다.

Startup Background Color​

페인팅 진행시 밑바탕이 되는 색상을 설정하는 속성입니다. 페인팅 도중 변경 가능합니다.

인터페이스​

Painting 2D​

Painting 2D 툴는 페인팅 모드와 동기화 되는 에디터 형태의 툴입니다.

Material 그룹​

페인팅 모드의 Material 그룹은 페인팅을 진행할 머티리얼을 선택하거나 씬에서의 표현에 관련한 속성을 모아 놓은 그룹입니다.

Layer 그룹​

페인팅 모드의 Layer 그룹은 레이어를 관리하고 편집할 수 있는 툴과 레이어가 표시된 그룹입니다.

레이어 마스크​

레이어 마스크는 레이어의 특정 영역을 마스킹하는 속성입니다.

Group 레이어​

Group 레이어는 레이어를 통합해서 관리하는 속성의 레이어입니다.

Paint 레이어​

Paint 레이어는 칠하는 방식으로 맵소스를 적용하는 레이어입니다.

Fill 레이어​

Fill 레이어는 맵소스의 값 자체를 모델 전체에 덮는 방식의 레이어 입니다.

Sticker 레이어​

Sticker 레이어는 이미지를 표면에 붙이듯이 표현하는 속성의 레이어입니다.

Properties 그룹​

Properties 그룹은 툴이나 레이어 등 선택한 기능에 따라 내용이 달라지는 영역으로 선택한 기능에 대한 속성이 표시됩니다.

Map Source 그룹​

Map Source 그룹은 텍스처의 해상도, 맵소스의 포맷 등을 설정하며, 맵소스 추가/삭제와 같은 맵소스 관리에 대한 속성을 모아놓은 그룹입니다.
