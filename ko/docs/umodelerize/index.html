일반 메쉬에서 UModeler X로 변환

UModelerize는 UModeler X로 제작되지 않은 모델을 UModeler X에서 편집 가능한 상태로 변환하는 기능입니다.

해당 기능은 UModeler X의 모델링 모드나 다른 모드에서 직접 사용하는 것이 아니라, 변환하려는 모델을 Unity 에디터의 Hierarchy 탭에서 선택 후 우클릭하여 나오는 메뉴에서 실행합니다.

사용 방법​

  1. 변환하려는 모델을 Unity 에디터의 Hierarchy 탭에서 선택합니다.
  2. 선택한 모델에 대해 우클릭을 합니다.
  3. 우클릭으로 나온 메뉴에서 "UModeler X > UModelerize"를 선택합니다.
  4. 선택한 모델이 UModeler X에서 편집 가능한 상태로 변환됩니다.
팁

UModelerize Merge Vertices​

모델에 따라서는 메시의 폴리곤이 분리되는 현상이 발생할 수 있습니다.

이런 경우, UModeler X > UModelerize Merge Vertices를 이용하여 같은 위치에 있는 버텍스들을 합쳐주면 해결할 수 있습니다.
