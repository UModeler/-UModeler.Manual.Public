UModeler X 라이선스 모델, 기술 로드맵 및 최신 업데이트

2024년 7월 26일 · 약 7분
황재식
Head of UModeler Development

안녕하세요,

7월의 막바지에 새로운 블로글로 인사드립니다. 지난 한 달 동안 개발뿐만 아니라 비즈니스 모델에도 많은 업데이트가 있었습니다. 이번 블로그 글을 통해 그 소식을 전해드리겠습니다.

UModeler X 라이선스 모델​

지난 9월부터 지금까지 UModeler X는 베타 테스트를 하면서 4,200명 가량의 사용자들로부터 다양한 피드백을 받을 수 있었고 덕분에 UModeler X는 한층 더 강력하고 안정적인 툴로 거듭났습니다. 그리고 마침내 UModeler X의 라이선스 모델이 어느 정도 확정되었습니다. 새로운 라이선스 모델이 적용된 후에도 기존 베타 버전에서 제공하는 다양한 모델링, 리깅, 페인팅 그리고 AI 텍스처링 툴은 앞으로도 Basic 라이선스로 무료로 제공할 예정입니다.

Plus 라이선스는 Basic 라이선스의 모든 기능을 포함하며, UModeler의 핵심 기능과 주요 피처를 탑재하여 기존 UModeler를 완전히 대체하는 것을 목표로 하고 있습니다. 기존 UModeler 사용자에게는 업그레이드 비용만으로 Plus 라이선스를 구매할 수 있는 혜택을 제공할 예정입니다. 이 Plus 라이선스는 2024년 9월에 얼리 버드 버전으로, 2024년 11월에 정식 버전으로 유니티 에셋 스토어에 출시될 계획입니다.

Pro와 Team 라이선스는 구독 모델로 내년 초에 공개될 예정이고 Plus의 모든 기능과 더불어, UModeler X의 최신 기능을 가장 빠르게 사용할 수 있는 혜택을 제공할 것입니다.

UModeler X 라이선스 정보는 아래의 FAQ 페이지를 참고해주기 바랍니다.

기술 로드맵​

라이선스 모델 확정과 더불어 기술 로드맵도 어느 정도 정해졌습니다.

앞으로 출시될 모든 라이선스 모델에 현재 사용 중인 Beta 버전의 텍스처 파라미터 툴과 Face Color 툴이 포함될 예정입니다.

Plus 라이선스에는 기존 UModeler의 핵심 기능인 스케치 기반 모델링, 핫스팟 텍스처링, Array 툴, Follow 툴 등이 추가될 예정입니다.

Pro와 Team 라이선스에는 3D 콘텐츠 제작을 한 단계 업그레이드할 AI 텍스처링 2.0, Curve 에디터, Pelt 언랩 등 고급 기능이 포함될 예정입니다. 또한, 새로운 기능이 지속적으로 추가될 것입니다.

2025년 1분기까지의 기술 로드맵은 아래와 같습니다. 다음 기능 목록은 상황에 따라 변경될 수 있습니다.

  • 기술 로드맵 (2024년 7월~2025년 3월)
    • 2024년 7월 - Beta 버전
      • 텍스처 파라미터 툴
      • Face Color 툴
    • 2024년 9월 - Plus 라이선스 Early Bird 버전
      • Array 툴 (UModeler의 Clone 툴과 유사)
      • Export 툴 (.OBJ, .STL, .PLY 지원)
      • PushPull 툴
      • 스케치 기반 모델링
    • 2024년 11월 - Plus 라이선스 정식 버전
      • Follow 툴
      • 핫스팟 텍스처링
      • 멀티 채널 UV 에디팅
      • 등등
    • 2025년 2월 - Pro, Team 라이선스
      • AI 텍스처링 2.0
      • Curve 에디터
      • Pelt 언랩
      • Voxel 에디터
      • Plan 에디터
      • 등등

업데이트 소식​

지난 한달 동안 UModeler X의 주요 업데이트 소식을 전해드리겠습니다.

버전 0.19.20부터 0.19.24까지 레벨 디자인을 도와줄 다양한 기능들이 추가되었습니다.

Preference 윈도우에서 설정할 수 있는 새로운 기능​

  • Create As Static Object : 해당 옵션을 통해서 UModelerX Object가 기본적으로 Static으로 생성되도록 할 수 있습니다.
  • Static Option : Create As Static Object 옵션이 활성화된 상태에서 생성되는 UModelerX Object의 Static 옵션을 설정할 수 있습니다.
  • Always Enable Grid Snap : 해당 옵션을 통해서 UModelerX Object가 선택되었을 때, Grid Snapping 기능을 항상 활성화 시킬 수 있습니다.
  • Create Under Root : 해당 옵션을 통해서 UModelerX Object가 부모 오브젝트를 가지지 않는 상태로 생성될 수 있습니다.

기본 도형 툴에 Create Shape as a New Object 속성 추가​

새로운 기본 도형을 만들 때마다 새로운 오브젝트에 생성될 수 있도록 합니다.

Snap Panel 윈도우​

Grid 스냅과 Rotate 스냅 사이즈를 미리 등록하고 손쉽게 원하는 스냅 사이즈를 사용할 수 있습니다.

그 밖에 많은 부분들이 개선되었습니다. 보다 자세한 업데이트 소식은 온라인 매뉴얼의 릴리즈 노트를 참고하시길 바랍니다.
