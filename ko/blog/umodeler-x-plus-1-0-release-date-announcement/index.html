UModeler X Plus 1.0 출시일 안내

2024년 11월 12일 · 약 4분
황재식
Head of UModeler Development

안녕하세요, UModeler X Plus 사용자 여러분

기다려주신 UModeler X Plus 1.0 정식 버전의 출시일을 11월 20일로 확정해 알려드립니다. 9월 얼리버드 버전 출시 이후, 정식 버전을 위해 저희 팀이 전력을 다해 개발에 매진해 왔습니다.

다만 한 가지 아쉬운 소식을 전해드려야 할 것 같습니다. 9월에 약속드렸던 스케치업 스타일 모델링 기능을 20일까지 완벽하게 구현하기가 어렵게 되었습니다. 새로운 UModeler X 아키텍처에서 이 기능을 완벽히 구현하는 데 예상보다 많은 기술적 과제가 있었기 때문입니다. 이에 1.0 버전 출시 후 지속적인 업데이트를 통해 이 기능을 개선해 나갈 계획입니다.

특히 Line 툴의 경우, 폴리곤 컷팅에 특화된 기능적 성격을 고려하여 Add 그룹으로 이동하면서 Knife 툴이라는 보다 직관적인 이름으로 변경됩니다. 또한 스케치업 스타일의 모델링을 위해 Drawing 그룹에 새로운 Poly Line 툴이 추가될 예정입니다. 핫스팟 텍스처링의 경우도 기존 UModeler 수준으로 우선 제공한 뒤, 추가 기능들을 단계적으로 구현해 나갈 예정입니다.

대신 이러한 아쉬움을 달래드리고자, 저희는 몇 가지 새로운 기능들을 준비했습니다:

  • 10월에 공개된 노멀 페인팅
  • Humonoid Avatar 지원
  • 한층 개선된 Display/Debug/Snap Settings UI
  • 오프라인 사용 지원

이러한 새 기능들이 여러분의 3D 콘텐츠 제작 워크플로우를 더욱 효율적으로 만들어줄 것이라 확신합니다.

스케치업 스타일 모델링과 핫스팟 텍스처링은 최우선 과제로 삼아 연내 완성을 목표로 개발하겠으며, 개발 진행상황도 수시로 공유드리도록 하겠습니다.

다음은 UModeler X Plus 1.0에 포함되는 주요 기능들입니다

  1. SketchUp Style Modeling 버전 0.7
  2. Hotspot Texturing 툴
  3. Follow 툴
  4. Array 모디파이어와 툴
  5. 노멀 페인팅
  6. .OBJ, .STL, .PLY Exporter
  7. Humonoid Avatar 지원
  8. Multi UV
  9. UModeler X 3D 모델 데이터의 씬저장 지원
  10. 개선된 Debug, Display Settings UI
  11. Snap Panel 윈도우에 World, Increment 옵션 추가
  12. 오프라인 사용 지원

1.0 출시 이후 로드맵

  • SketchUp Style Modeling ver 1.0
    • 안정성 향상
    • 외부 폴리곤과 내부 홀 연결 기능 지원
    • 등등
  • Hotspot Texturing 추가 기능
    • 트림시트
    • 그룹 단위 핫스팟
  • 개선된 Pivot Gizmo 시스템

11월 20일에 정식 버전 출시 소식으로 다시 찾아뵙겠습니다.
