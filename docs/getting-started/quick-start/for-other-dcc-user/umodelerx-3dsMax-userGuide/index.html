3dsMax 유저를 위한 UModelerX

기본적인 차이

1. 단위​

UModelerX는 유니티 엔진의 기본 단위인 미터 m 를 쓰고 있습니다. UModelerX에 표시되는 길이와 크기에 대한 값은 모두 미터 기준으로 표시되어있습니다.

2. UI​

1. 작업 공간​

UModelerX 오브젝트를 편집할 수 있는 공간입니다.

UModelerX는 메쉬 편집을 위해 다른 창을 사용하지 않고, 제작 중인 3D 콘텐츠와 같은 환경인 씬에서 그대로 편집할 수 있습니다.

2. 툴바​

씬뷰 상단에 있는 아이콘 툴바를 통해 여러 엘러먼트를 선택할 수 있고 UModelerX 오브젝트를 편집할 수 있습니다.

이에 대한 자세한 내용은 [Element 그룹] 페이지를 참고하세요.

3. UModelerX 컴포넌트​

여러 툴과 그룹으로 이루어진 인스펙터 내 UModelerX 컴포넌트 입니다.

4. UV 에디터​

UModelerX에서 UV를 편집할 수 있는 에디터 입니다. 별도의 탭으로 유니티 엔진 내에 배치하거나 분리할 수 있습니다.

빠른 사용자 가이드인 [UV 에디터 시작하기] 페이지를 참고하세요.

3. UV​

3dsMax와는 다르게 UModelerX 오브젝트의 UV는 직접 펼치거나 [UModelerize] 된 오브젝트가 아니라면 펴지지 않은 상태입니다.

펴지지 않은 상태에서는 UV 좌표가 Local 기준으로 맞춰져 있기 때문에 이미지나 그리드가 표현된 텍스처를 바탕으로 모델링을 하기 수월합니다.

4. 새 오브젝트 생성​

3dsMax의 경우 지오메트리를 생성 후 편집 가능한 상태(Convert to Editable Poly, Edit Poly 등)로 변환하여 편집 해줘야 합니다.

UModelerX는 New Object를 이용하여 새 오브젝트를 생성할 수 있고 [UModeler X 시작하기] 페이지를 참고하여 여러 오브젝트 생성 방법을 알아볼 수 있습니다.

모디파이어​

3dsMax의 모디파이어 기능은 다양하지만 실제 유니티엔진에서 사용을 하지 못하거나, 변환해야 되는 기능들이 많습니다.

UModelerX의 경우 모델링 기능들 중 가장 많이 쓰이거나 효율적인 기능들을 툴과 모디파이어로 제작하여 제공합니다.

이에 대한 자세한 내용은 [모디파이어 시스템] 페이지를 참고하세요.

UModelerX의 특징

오브젝트 편집 벗어나기​

3dsMax에서 오브젝트 편집을 벗어나기 위해 Selection 탭에 활성화된 버튼을 한번 더 눌러 편집상태에서 벗어나지만, UModelerX는 상단 [Object 툴]: 단축키 Shift+4 을 누르거나 하이어라키 탭에서 다른 오브젝트를 눌러 편집 상태에서 벗어날 수 있습니다.

UModelerize 툴​

3dsMax에서 만들었거나 UModelerX가 아닌 다른 프로그램에서 제작된 모델을 UModelerX를 이용하여 편집하려면 [UModelerize]를 이용하여 유모델러 오브젝트로 변환해야 합니다.

툴의 옵션 설정​

툴 중에는 동작 옵션을 설정할 수 있는 툴이 있습니다. 클릭을 통해 툴의 기능이 적용될 때 옵션에 따라 같은 툴이라도 동작이 달라집니다.

툴 버튼 오른쪽 하단을 보면 밝은 회색의 삼각형이 있는데 해당 표시가 있는 툴 버튼을 **Alt+좌클릭** 하게 되면 툴이 바로 작동하는 것이 아닌 툴의 옵션창이 띄워집니다.

변경된 옵션값과 옵션창은 따로 닫거나 변경하지 않는 한 계속 유지되기 때문에 원할 때마다 빠르게 옵션을 변경하여 사용할 수 있습니다.

Rigging 모드​

Rigging 모드는 애니메이션을 위한 모드입니다. 이를 통해 유니티 내에서 메쉬의 버텍스 웨이트 할당과 함께 웨이트 편집을 할 수 있습니다.

이에 대한 자세한 내용은 [Rigging 모드] 페이지를 참고하세요.
