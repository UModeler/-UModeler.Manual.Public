Preference

The Preference Window is where you can configure various options to make some of the functionality of UModeler X easily accessible and usable.

How to use​

It can be accessed by navigating to Edit > Preferences > UModeler X in the Unity Editor.

Properties

Editable Mesh​

Auto Unwrap Shape : Sets the shape for auto unwrapping

Auto Generate Mesh Collider : Automatically generates a mesh collider

UModelerize Merge Vertices : Merges vertices during UModelerize

Create As Static Object : Creates the object as a static object

Static Option : Sets the static options

Always Enable Grid Snap : Always enables grid snapping

Create Under Root : Creates under the root object

White Box Mode : Sets the white box mode

Scene Save Asset : Sets to save UModelerX object data in scene subfolder

Inspector Menu Type : Sets the display type of the Inspector menu

  • Icon + Text, Icon, Text

Modeling​

Vertex Size : Sets the size of the vertices

Vertex Color : Sets the color of the vertices

Edge Color : Sets the color of the edges

Selected Color : Sets the color of the selected elements

Highlight Color : Sets the color of the highlighted elements

Seam Color : Sets the color of the seams

Deform Brush Color : Sets the color of the deform brush

Wireframe Thick : Sets the thickness of the wireframe

Selected Wireframe Thick : Sets the thickness of the selected wireframe

Extrude Modifier Key : Sets the modifier key to use during the Extrude operation

Rigging​

Rigging Bone Color : Sets the color of the rigging bones

Rigging Selected Bone Color : Sets the color of the selected rigging bones

Rigging Weight Low Color : Sets the color for low rigging weights

Rigging Weight High Color : Sets the color for high rigging weights

Rigging Weight Brush Color : Sets the color for brush rigging weights

Rigging Weight RGB Color : Sets the RGB color for rigging weights

Lightmap​

Auto Generate Lightmap UV : Automatically generates lightmap UVs

UnwrapParam Angle Error : Sets the maximum allowable angle distortion during unwrapping

UnwrapParam Area Error : Sets the maximum allowable area distortion during unwrapping

UnwrapParam Hard Angle Error : Sets the angle threshold for creating seams during unwrapping

UnwrapParam Pack Margin Error : Sets the padding between UV islands to prevent overlap

Material​

Default Material : Sets the default material

Texture : Sets the texture

Smoothness : Sets the smoothness of the surface

Tiling : Sets the tiling frequency of the texture

Color : Sets the color

UV Editor​

UV Editor Menu Type : Sets the display type of the UV editor menu

  • Icon + Text, Icon, Text

Is Settings Visible : Shows the settings

Background Color : Sets the background color

Fluid Grid Line Color : Sets the color of the fluid grid lines

Vertex Size : Sets the size of the vertices

Vertex Color : Sets the color of the vertices

Edge Color : Sets the color of the edges

Face Color : Sets the color of the faces

Selected Color : Sets the color of the selected elements

Highlight Color : Sets the color of the highlighted elements

Shared Color : Sets the color of the shared elements

Seam Color : Sets the color of the seams

Wireframe Thickness : Sets the thickness of the wireframe

Selected Wireframe Thickness : Sets the thickness of the selected wireframe

Painting​

Show Axis : Shows the axis

Show Grid : Shows the grid

Show Wireframe : Shows the wireframe

Brush Size : Sets the size of the brush

Wireframe Line Color : Sets the color of the wireframe lines

Wireframe Thickness : Sets the thickness of the wireframe

Thumbnail Type : Sets the size of the thumbnails

  • Large, Small

Startup Background Color : Sets the background color of the workspace or canvas

Startup Resolution : Sets the initial resolution of the workspace

Startup Map Source Template : Sets the default map or texture template

Auto Save Textures : Automatically saves textures

Painting 3D / GenAI​

Num Displayed Item : Sets the number of displayed items

Texture Dilation Size : Sets the size of the texture dilation beyond the UV boundaries

Depth Bias : Sets the distance between the brush and the surface to be painted

Save All Generated Result : Saves all generated results

Save Result Path : Sets the save path

Show Completion Notification Popup : Shows a completion notification popup

Display​

Size Overlay : Shows the size of the selected object (X, Y, Z dimensions)

Ruler : Shows the length of the edges

EdMesh Info : Shows the EdMesh information

Scene Toolbar : Shows the scene toolbar

Toolbar Position : Sets the position of the toolbar (X, Y)

Commentary : Visually displays the name, shortcut key, or related information of the executed action

Export​

Create Folder : Creates a folder

Export Material : Exports the material

Export Vertex Color : Exports the vertex color

Optimize UVs : Optimizes the UVs

Export Unit : Sets the unit for export

  • Meter, Centi Meter, Milli Meter

Export Polygon Type : Sets the polygon type for export

  • Triangles, Quad, Polygons

Export Stl Format : Exports in STL format

  • Ascii, Binary
