Preference

UModeler의 옵션을 설정할 수 있는 Preference Window에 대한 설명입니다

사용 방법

Unity 에디터에서 Edit - Preference - UModeler X 순서로 Preference Window에 진입할 수 있습니다

속성

Display​

Scene에 보이는 표시정보를 활성화하거나 해지할 수 있습니다

Size Overlay​

현재 편집 중인 오브젝트의 사이즈를 표시해 줍니다

Size Overlay OFF

Size Overlay ON

Edmesh Info​

Edmesh의 각 엘러먼트 정보를 Scene에 표기해 줍니다

Edmesh Info OFF

Edmesh Info ON

Material​

UModeler 오브젝트에서 기본적으로 사용하는 Material을 세팅할 수 있습니다

Texture​

UModeler 오브젝트에서 사용할 텍스처를 선택합니다

◎ 버튼을 통해 텍스처를 선택할 수 있습니다

Smoothness​

Smooth 값을 0~1사이로 설정할 수 있습니다

Tiling​

선택한 텍스처를 타일처럼 반복 배치할 수 있습니다

X : X축으로의 텍스처 비율을 정할 수 있습니다

Y : Y축으로의 텍스처 비율을 정할 수 있습니다

Color​

Material의 기본 색상을 변경할 수 있습니다

직접 색상을 선택하거나 스포이드를 통해 색을 추출할 수 있습니다

Ed Mesh​

Scene에서 보이는 Mesh의 엘러먼트 색상, 크기 등을 설정할 수 있습니다

Vertex Size​

Scene에서 보이는 버텍스의 크기를 조절할 수 있습니다

Vertex Color​

Scene에서 보이는 버텍스의 색상을 변경할 수 있습니다

Edge Color​

Scene에서 보이는 엣지의 색상을 변경할 수 있습니다

Selected Color​

선택된 엘러먼트의 색상을 변경할 수 있습니다

Highlight Color​

하이라이트 된 엘러먼트의 색상을 변경할 수 있습니다

Wireframe Thick​

Scene에서 보이는 기본 엣지의 두께를 변경할 수 있습니다

Selected Wireframe Thick​

Scene에서 보이는 선택된 엣지의 두께를 변경할 수 있습니다

UV Editor​

UV Editor 에서 보이는 엘러먼트의 속성을 변경 할 수 있습니다

Background Color​

UV에디터 내의 바탕색을 변경할 수 있습니다

Fixed Grid Line Color​

고정 그리드의 색상을 변경할 수 있습니다

Fluid Grid Line Color​

가변 그리드의 색상을 변경할 수 있습니다

Vertex Size​

UV에디터 내의 버텍스 사이즈를 변경할 수 있습니다

Vertex Color​

UV에디터 내의 버텍스 색상을 변경할 수 있습니다

Edge Color​

UV에디터 내의 엣지 색상을 변경할 수 잇습니다

Face Color​

UV에디터 내의 페이스 색상을 변경할 수 있습니다

Selected Color​

UV에디터 내의 선택된 엘러먼트의 색상을 변경할 수 있습니다

Highlight Color​

UV에디터 내의 하이라이트 된 엘러먼트의 색상을 변경할 수 잇습니다

Shared Color​

선택된 엘러먼트와 연결되어 공유되는 엣지의 색상을 변경할 수 있습니다

Seam Color​

Seam이 적용된 엣지의 색상을 변경할 수 있습니다

Wireframe Thickness​

와이어 프레임의 굵기를 변경할 수 있습니다

Selected Wireframe Thickness​

선택된 와이어 프레임의 굵기를 변경할 수 있습니다

Debug​

Debug 설정을 변경할 수 있습니다

Display Inspector UI​

인스펙터에서 유모델러 UI를 보여줍니다

Display Scene UI​

Scene 상단의 유모델러 UI를 보여줍니다

Debug Mode​

디버그 모드를 켜고 끕니다
