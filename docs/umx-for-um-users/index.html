UModeler X for UModeler users

UModeler enabled fast modeling with its sketch-based smart modeling and concise, quick features. However, UModeler X takes this a step further by providing all the features needed for 3D content creation.

UModeler X includes everything from a modeling mode for creating shapes, UV editing, a rigging mode for animation, to a painting mode for coloring and expressing distinctive materials.

This guide explains how the familiar features in UModeler have changed in UModeler X, and how to use the new features.

UModeler X offers enhanced performance and new features compared to UModeler.

UModeler X provides a UV unwrapping feature using the LSCM (Least Squares Conformal Maps) algorithm. This allows users to generate UV coordinates quickly and efficiently. By using the LSCM tool within the UV Editor, it calculates the least squares distance to produce the optimal UV layout.

Editing much more polygonsâ€‹

UModeler X provides enhanced polygon editing features along with optimized performance, allowing for the display and editing of more polygons.

Thanks to this, it is easier to create complex and detailed models while managing polygons more efficiently.

It is effective for creating characters that require curved and smooth modeling, or backgrounds requiring complex shapes, and thanks to the optimized performance, you can experience a smoother working environment compared to UModeler.

2. Simultaneous use of UModeler and UModeler Xâ€‹

UModeler X can be used together with UModeler. Each version has its own advantages, so you can choose and use them appropriately according to the requirements of the project.

In the initial stages, you can utilize the simplicity of UModeler for prototyping, level design, and quick background creation.

Later, in the stages where detailed work is needed, it is possible to complete models using the various features of UModeler X, including modeling, modifiers, rigging tools, and layer-supported painting modes.

In this way, both versions can be used simultaneously within a single project, allowing users to freely choose according to their production workflow.
