Snap Panel

The Snap Panel is a window that allows you to adjust the snap intervals for movement and rotation.

Select "Tools > UModeler X > Snap Panel" from the top menu.

Alternatively, click the Grid and Settings button in the Tools section of the Overlay Menu to pop up the Snap Panel window.

Movement​

You can set the default snap intervals for movement.

  • Preset sizes: 0.0625, 0.125, 0.25, 0.5, 1, 2

Rotation​

You can set the default snap intervals for rotation.

  • Preset sizes: 15, 30, 45, 60, 75, 90

Settings​

  • Change Button Position: Drag the handle up or down to change the position of the snap interval buttons.
  • Add Custom Size Button: Add a custom snap interval in addition to the preset snap intervals.
  • Delete Snap Interval Button: Remove unnecessary snap interval buttons.

Reset​

You can reset all button lists and activated buttons to their initial state.
