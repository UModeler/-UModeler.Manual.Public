Rigging Mode Guide

The Rigging Mode in UModeler X is a mode that allows you to add, edit, and skinned bones for animation. In this mode, you can assign weights to vertices and edit weights to implement the movement of your model.

What is Rigging?​

Rigging is the process of creating a skeleton for a 3D model, and setting up how the model deforms as the skeleton moves. This process uses a weighting concept called Weights to determine how much the vertices reflect the movement of the bones.

In a nutshell, rigging is the process of giving a 3D model a bone that allows it to animate, and connecting the vertices so that when the bone moves, the vertices move together based on their weight values.

info

Weights​

In Rigging Mode, Weight stands for Weighted, and is a weight value that determines how much of the bone's movement is reflected in the vertices.

  • From left, weight values are 0.0, 0.5, 1.0

The values range from 0.0 to 1.0, where 0.0 means that the bone's movement will not affect that vertex at all, and 1.0 means that the bone's movement will completely affect that vertex. This allows you to control the movement of your 3D model.

Why do we need rigging?​

Without rigging, your 3D model will only exist in a static state, meaning it will be like an immobile statue.

Rigging gives you the flexibility to move characters or objects that need to move, allowing you to create animations.

info

Static and Skinned Meshes​

When you create and use 3D models in Unity, you use two types of meshes.

Static Meshes: A Static Mesh is an object that does not deform, meaning it is a mesh that does not animate. They are primarily used for non-moving objects such as backgrounds, buildings, and furniture.

  • Of course, the "universal" use of Static Meshes is for furniture and buildings, but if you need deformation within the object, you can create a Skinned Mesh.

Skinned Meshes: A Skinned Mesh is a 3D model that can be rigged and animated. They are used for moving objects such as characters and animals. Skinned meshes are connected so that their vertices move using bones and weight values, allowing them to be animated.

tip

How to identify Static and Skinned Meshes in Unity​

  1. select your 3D model.
  2. When selected, the Inspector window displays the model's information.
    • If you have a Mesh Filter component and a Mesh Renderer component:
      1. This is a Static Mesh.
      2. These two components are responsible for the mesh's information and settings for rendering.
    • If you have a Skinned Mesh Renderer component:
      1. This is a Skinned Mesh.
      2. This component supports animation and has the ability to deform the mesh based on bone and weight values.

This is how we distinguish and identify Static and Skinned Meshes in Unity.

Step-by-step rigging process​

  1. Prepare a UModeler X object that has been modeled and UV unwrapped.
  2. Enter Rigging Mode.
  3. Create a bone.
  4. Skinning it using the weight-related tools.
  5. Move or animate the bones to see the results.

Below is a description of each sequence.

1. Prepare a UModeler X object that has been modeled and UV unwrapped​

Modeling and UV unwrap are essential steps to complete before rigging. Modeling is the process of creating the shape of the 3D model, and UV unwrap is the process of correctly mapping textures to the surface of the 3D model. Only after these two processes are completed can the model be rigged.

  • Check out the documentation below to learn about modeling modes and the UV Editor.

📄️ Modeling Mode

📄️ UV Editor

2. Enter Rigging Mode​

Once modeling and UV unwrap is complete, enter Rigging Mode. Rigging mode provides various tools for rigging, such as creating bones and editing weights.

3. Create Bones​

Bones act as a skeleton to control the movement of a 3D model. In the process of creating bones, you need to set the position of the bones and the connection structure between them.

UModeler X provides the Build Bones tool to facilitate this process. With this tool, you can quickly configure the connection structure of the bones in the desired form.

📄️ Build Bones Tool

We also provide an Edit Bones Tool that allows you to make edits, such as modifying or adding bones that have already been created.

📄️ Edit Bones Tool

4. Proceed with Skinning​

Skinning is the process of associating bones with vertices. In this process, we assign a weight to each vertex to determine how much the movement of the bone will affect that vertex.

  • You can accomplish this process by using the Edit Weights tool. The following tools are available for editing weights.

📄️ Weight Tool

📄️ Weight Brush Tool

  • Here's a tool to assign weights automatically or to make it quick and easy.

📄️ Weight Transfer Tool

📄️ Auto Weight Tool

📄️ Envelope Tool

5. Checking the Results​

The final step in rigging is to move or animate the bones to see the results. This step allows you to verify that the rigging was done correctly and that the desired movement was achieved.

If you didn't get the results you wanted, you can modify the structure or weights of the bones to achieve the desired results.
