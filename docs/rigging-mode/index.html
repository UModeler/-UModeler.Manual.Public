Rigging Mode

The Rigging mode is for animating models, allowing you to assign weights to meshes and edit weights.

info

Weights​

The Weight we are talking about in rigging mode is a weighting value for how much the movement of the bones should be reflected in the vertices to make them move.

Values range from 0.0 to 1.0, where 0.0 means that moving the bone will not move the corresponding vertex, and 1.0 means that moving the bone will move the corresponding vertex with it.

Starting Rigging Mode​

To use the Rigging Mode, you first need to set up bones for the rigging work.

Root Bone​

This is the slot for selecting the bone that is positioned at the top of the hierarchy. The bone selected in this slot becomes the center point that controls the movement and rotation of the model.

Interface​

Active/Inactive​

Button to enable or disable rigging mode.

Clear Rigging Data​

Button to clear rigging related data currently applied to the object.

This is displayed when rigging mode is disabled via the Active/Inactive button.

Root Bone Slot​

This is the slot to set the top-level bone to apply to the selected UModelerX object.

Select the top-level bone object in the Hierarchy tab and click and drag to set it.

Bone Tree​

This is the area where you can see the tree structure of the bones assigned in the Root Bone slot.

The Bone Tree shows all connected bones, regardless of weight assignment, and allows for easy bone selection.

Bind Bone List​

This area displays bones that have been assigned weights and are associated with a vertex.

Like the Bone Tree area, it allows for easy bone selection.

Tools Group​

The Tools group in Rigging Mode is a collection of tools to modify weights or adjust bones.

Weight Tools​

The Weight tool is a tool that allows you to assign weights to selected vertices.

Weight Brush Tool​

The Weight Brush tool allows you to assign a vertex's weight to a selected bone by brushing it.

Reset Poses Tool​

The Reset Poses tool resets the position and rotation of a moved bone, returning it to its original default pose.

Rebind Poses Tool​

The Rebind Poses tool is used to reset the position and rotation of a bone.

Weight Transfer Tool​

The Weight Transfer tool is a tool that copies the weight applied to another object to the current object.

Envelope Tool​

The Envelope Tool is a tool for editing an Envelope, which is an encapsulated volume that gives weight to vertices.
