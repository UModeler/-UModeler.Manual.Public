Mirror Object Tool

Mirror Object Tool is a feature that duplicates an object symmetrically based on a specified axis.

How to Use​

  1. Select the object you want to mirror.
  2. Activate the Mirror Object Tool.
  3. CAdjust the properties to appropriate values.
  4. The object will be duplicated symmetrically according to the selected properties.

Properties​

Mirror Origin Type​

Sets the axis reference to the object's Local coordinates.

Local Sets the axis reference to the object's Local coordinates.

World Sets the axis reference to the World coordinates.

Mirror Axis​

Sets the axis reference to the World coordinates.

X Uses the X-axis as the symmetry reference.

Y Uses the Y-axis as the symmetry reference.

Z Uses the Z-axis as the symmetry reference.

Distance​

A property that adjusts the distance between the center of symmetry and the mirrored object.

Angle​

A property that modifies the rotation value of the axis.

Flip​

A property that reverses the direction of the mirrored axis.
