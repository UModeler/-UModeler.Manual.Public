Custom Coordinate Tool

Custom Coordinate Tool is a tool for setting up Custom Coordinate System in UModelerX.

note

Custom Coordinate System

You can check the Custom Coordinate System through the UModelerX Toolbar.

How to Use​

  1. Select the Origin Point among the vertices of the chosen object.
  2. Select the X-Axis direction among the other vertices of the object.
  3. Select the Y-Axis direction among the other vertices of the object.
  4. Check if the Custom Coordinate System is correctly set in the UModelerX Coordinate System Toolbar.
note

If the x-axis or y-axis is not set correctly, the process may not proceed.
