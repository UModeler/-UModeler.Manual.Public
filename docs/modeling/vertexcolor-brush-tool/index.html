Vertex Color Brush 툴

Vertex Color Brush 툴은 버텍스에 버텍스 컬러를 적용하는 브러시 형태의 툴입니다.

참고

버텍스 컬러(Vertex Color)​

버텍스 컬러는 버텍스에 저장되는 색상 정보**입니다. 버텍스 간의 색상이 자동으로 보간되며, 다양한 용도로 활용될 수 있습니다.

사용 방법​

  1. Vertex Color Brush 툴을 실행합니다.
  2. View Vertex Color 속성을 활성화하면, 오브젝트의 버텍스 컬러가 표시됩니다.
  3. Brush Color 속성을 클릭하여 원하는 색상으로 변경합니다.
  4. 마우스 커서를 오브젝트의 버텍스 위로 이동한 후, 클릭드래그하여 버텍스 컬러를 칠합니다.

속성​

VertexColor​

View Vertex Color​

오브젝트의 버텍스 컬러를 표시하는 속성입니다.

Vertex Color​

버텍스 컬러를 지정할 수 있는 속성입니다.

Brush Data​

브러시의 크기와 세기를 조절하는 속성들입니다.

Size Type​

브러시 크기의 기준을 선택하는 속성입니다.

Pixel

브러시 크기를 픽셀 기준으로 정하는 속성입니다.

Brush Radius Pixel 속성에서 크기를 설정할 수 있습니다.

Distance

브러시 크기를 월드 상의 크기를 기준으로 정하는 속성입니다.

Brush Radius Distance 속성에서 크기를 설정할 수 있습니다.

Distance Centimeter

브러시 크기를 월드 상의 크기를 기준으로 정합니다. 단위는 센티미터입니다.

Brush Radius Distance 속성에서 크기를 설정할 수 있습니다.

Brush Radius Distance​

월드 상의 크기 기준의 브러시 크기 설정하는 속성입니다.

Size Type 속성이 Distance, Distance Centimeter로 되어있어야 사용 가능합니다.

Brush Radius Pixel​

픽셀 기준의 브러시 크기 설정하는 속성입니다.

Size Type 속성이 Pixel로 되어있어야 사용 가능합니다.

Brush Strength​

브러시 세기에 대한 속성입니다.

Fall Off​

브러시 모양을 결정하는 그래프 형태의 속성입니다. Fall Off 옆 그래프를 클릭하여 편집할 수 있습니다.

Fall Off 형태 별 선택 모양 변화 예시입니다.
