Room Tool

The Room tool is a tool for creating room-shaped models.

How to use​

The basic shape creation tools in the Primitive Shapes group can create shapes in two ways.

  1. Click-Drag to create, edit
  2. Create and edit using One Click Build

Create and edit by clicking and dragging​

  1. Select the Room tool.
  2. Create a Bottom of the desired size by click-dragging it with the mouse anywhere in the scene.
  3. Once the base is created at the desired size, release the click and move the mouse up to generate a mesh at the desired height.
  4. When you reach the desired height, click again to finish creating the mesh.
  5. Adjust the thickness of the wall by adjusting the Thickness in the Room tool's properties.
    • If you enable the XRay Mode, the face will appear translucent, making it easier to adjust.
  6. Adjust the remaining properties of the Box Tool and select the element, or press another tool to finish.
note

You can continue to fine-tune with the Room Tool's Properties until you select an element or press another tool to finalize.

One Click Build and Edit via Properties​

  1. Select the Room tool.
  2. Enter the desired values in the properties of the Room Tool.
  3. Press the One Click Build button to add the shapes to the scene.
  4. Make any additional adjustments to the Room Tool's properties and select an element, or press another tool to finalize.
note

You can continue to fine-tune with the Room Tool's Properties until you select an element or press another tool to finalize.

Properties​

Width​

This property sets the horizontal size of the generated room shape.

It is in the direction corresponding to the X axis relative to the object's base, with units in meters.

Height​

This property sets the height of the generated room shape.

It is in the direction corresponding to the Y axis relative to the object's base, with units in meters.

Depth​

This property sets the Vertical size of the generated room shape.

It is in the direction corresponding to the Z axis relative to the object's base, with units in meters.

Thickness​

A property that sets the thickness of the walls of the generated room shape.

Display​

This property allows you to choose how the mesh is displayed.

XRay Mode​

Displays faces as translucent.
