Cone Tool

Cone Tool is a tool for creating shapes shaped like a cone.

How to use​

The basic shape creation tools in the Primitive Shapes group can create shapes in two ways.

  1. Click-Drag to create, edit
  2. Create and edit using One Click Build

Create and edit by clicking and dragging​

  1. Select the Cone Tool.
  2. Create the base of the cone by click-dragging the desired location in the scene.
  3. Once the base is created at the desired size, release the click and move the mouse up to adjust the height of the cone.
  4. When the desired height is reached, click again to finish creating the cone.
  5. Adjust the Cone Tool's properties and select an element, or press another tool to finish.
note

You can continue to fine-tune with the Cone Tool's Properties until you select an element or press another tool to finalize.

One Click Build and Edit via Properties​

  1. Select the Cone Tool.
  2. Enter the desired value in the properties of the Cone Tool.
  3. Press the One Click Build button to add the shape to the scene.
  4. Make any additional adjustments to the Cone Tool's properties and select the element, or press another tool to finish.
note

You can continue to fine-tune with the Cone Tool's Properties until you select an element or press another tool to finalize.

Properties​

Radius​

This property sets the radius of the cone being created.

Height​

This property sets the height of the generated cone.

Sides​

Property that sets the number of pieces that are split for the perimeter of the generated cone.

Display​

This property allows you to choose how the mesh is displayed.

XRay Mode​

Displays faces as translucent.
