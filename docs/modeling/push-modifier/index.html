Push Modifier

The Push modifier is a modifier used to push faces in the normal or opposite direction.

How to Use​

  1. Press the Add Modifier button.
  2. Select the Push Modifier.
  3. Change the Push Size property.
  4. the faces will be spaced apart and become larger or smaller.

Properties​

Push Size​

Property for how far apart the faces should be. If 0, there is no change.
