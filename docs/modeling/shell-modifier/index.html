Shell 모디파이어

개요

Box 툴은 기본적인 형태의 박스를 생성하는 툴입니다.

사용 방법

One Click Build와 속성을 통한 생성, 편집​

  1. Scene 탭 상단의 New Object 버튼을 클릭합니다.

  2. UModelerX 오브젝트 생성과 함께 Inspector 탭에 UModelerX 컴포넌트가 추가되며 생성이 가능한 기본 도형을 볼 수 있습니다. 여러 툴 중 Box 툴을 선택 합니다.

  3. Box 툴 선택과 함께 UModelerX 컴포넌트에 여러 속성과 버튼이 나옵니다.

  4. One Click Build 버튼을 클릭하면 Scene 탭에 구가 추가됩니다.

  5. 카메라 앞에 생성됨과 동시에 속성이 모디파이어 리스트 하단에 보여지게 됩니다.

  1. Width(가로) , Depth(세로) , Height(높이) , Width Segs(가로 면분할 수) ,Depth Segs(세로 면 분할 수) , Height Segs(높이 면 분할 수) 에서 원하는 수치에 맞게 입력하기 때문에 정확한 수치로 만들어 줄 수 있습니다.
참고

바로 에디팅을 시작하고 싶다면 Confirm 버튼을 클릭하면 됩니다.

  1. Width(가로) , Depth(세로) , Height(높이) 는 수치단위가 미터로 계산이 되서 1 ,1, 1 로 수치를 정할경우 가로, 세로 , 높이가 1미터의 박스 모디파이어가 만들어집니다. Width Segs(가로 면 분할 수) ,Depth Segs(세로 면 분할 수) , Height Segs(높이 면 분할 수) 경우 2,2,2의 수치로 정할경우 각 면마다 두 번 나눠지게 되어 4개의 면으로 분할이 됩니다.
  1. 원하는 속성 수치를 입력한 후 Confirm을 클릭하여 마무리합니다. 취소하고 싶을때 Cancel 을 클릭하면 사라지게 됩니다.

클릭드래그를 통한 생성, 편집​

  1. 새로운 모디파이어 버튼을 클릭합니다.
  2. Ed Mesh 모디파이어 생성과 함께 선택창이 생기는데 그중에서 Box 툴 을 선택 합니다.
  1. 카메라가 보이는 위치 앞에 자동 생성된 곳에 좌클릭한 상태에서 첫번째 방향으로 드래그 한 후 바로 두번째 방향으로 드래그 하여 밑면을 그려줍니다.
  2. 밑면을 그려준 후 마지막 위치점에서 좌클릭을 해제 한 후 마우스를 위로 이동을 시켜 높이를 정한 다음 마우스 왼쪽 버튼을 클릭한 후 마무리시킵니다.
  1. 생성 되어도 Confirm 버튼을 누르기 전에는 속성값으로 만들수 있습니다.
  1. 마우스로 드래그하여 만든 다음 Confirm버튼을 눌러 완성시킵니다.

상단 메뉴를 통한 Box모디파이어 생성​

위에 설명과 같이 아이콘에서 생성하는 방법이 있고 아래 이미지와 같이 상단메뉴에서 생성할수 있는 방법이 있습니다.

이미 생성된 Ed Mesh 모디파이어에서 Box 툴을 통해 도형 추가하는 방법​

박스 모디파이어가 생성된 상태에서 Ed Mesh 모디파이어에 있는 Primitive Shapes 그룹에는 여러가지 도형을 생성하기 위한 툴들이 있습니다. 박스 모디파이어를 선택하여 추가할 수가 있으며, ** 페이스 위에 추가로 **생성할 수 있고, 바닥에도 생성 할수 있습니다. 추가 생성 기능을 이용해서 다양한 형태의 모형을 제작할 수가 있습니다.

마우스 동작을 이용한 추가 생성방법​

  1. 박스 모디파이어를 생성한 상태에서 Ed Mesh 모디파이어를 선택하면 하단에 메쉬를 에디팅 하기 위한 툴 아이콘이 표시 됩니다. 여기서 다양한 기본 도형 툴 아이콘이 Primitive Shapes 그룹에 배치된 것을 볼 수 있습니다.
  2. Primitive Shapes 그룹의 Box 툴을 선택합니다.
  1. 이미 생성된 면에 원하는 위치에 커서를 대고 좌클릭 드래그로 밑면을 생성합니다.
  2. 밑면을 그린 후 좌클릭을 때고 면의 노멀 방향으로 커서를 움직이면서 원하는 높이를 만든 후 좌클릭으로 높이를 고정 시킵니다.
  3. Confirm과 Cancel 버튼 속성 아래 보이게 되고 이 때 속성을 변경하면 이미 생성된 형태를 바꿀 수 있습니다. Confirm을 클릭하면 생성이 되고 Cancel를 누르면 취소 됩니다.

Depth Segs​

Height Segs​

가로, 세로, 높이 각 축의 몇 개로 분할 할 것인지 지정합니다. 1 세그먼트(Segment)가 하나의 분할면을 나타내기 때문에 1 이상의 값을 가져야 합니다.

팁

Bend 툴이나 부드러운 곡선을 만들때 가장 많이 활용하는것이 Segments 입니다. 좀 더 디테일한 변형을 원할 경우 Segments 값을 늘려 설정 할 수 있습니다.
