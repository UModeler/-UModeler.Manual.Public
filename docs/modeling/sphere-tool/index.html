Sphere Tool

The Sphere tool is a tool that creates a basic form of a sphere.

How to use​

The basic shape creation tools in the Primitive Shapes group can create shapes in two ways.

  1. Click-Drag to create, edit
  2. Create and edit using One Click Build

Create and edit by clicking and dragging​

  1. Select Sphere Tools.
  2. Click and drag the mouse on the scene to create a sphere of the desired size.
  3. When it is the desired size, release the click to finish creating the mesh.
  4. Adjust the properties of the Sphere tool and select the element, or press another tool to finish.
note

You can continue fine-tuning with the properties of the Sphere tool until you select an element or press another tool to finalize.

Create, edit via One Click Build and Properties​

  1. Select the Sphere tool.
  2. Enter the desired values in the Properties of the Sphere tool.
  3. Press the One Click Build button to add a shape to the scene.
  4. Make any additional adjustments to the properties of the Sphere tool and select the element, or press another tool to finish.
note

You can continue to fine-tune the properties of the Sphere tool until you select an element or press another tool to finalize.

Properties​

Segments​

Allows you to set the number of segments of the generated sphere.

Rings​

Allows you to set the number of pieces that are split into the top and bottom of the generated sphere.

Radius​

Allows you to set the radius size of the generated sphere.
