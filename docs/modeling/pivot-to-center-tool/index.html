Pivot To Center 툴

개요​

Pivot To Center툴은 Ed Mesh 모디파이어들이 Pivot이 변경되는 경우가 있습니다. 프로젝트할 때 Pivot을 정리하는게 좋은데 가장 빠르게 Pivot 위치를 변경할수 있는 기능입니다.

참고

Pivot To Center 툴을 사용 할 시 오브젝트중심으로 활성화 할수 있도록 pivot으로 설정해야합니다. Center 표시는 단일 오브젝트나 여러오브젝트를 선택할시 선택한 오브젝트의 중심에 Pivot이 위치하게 되어있습니다.

속성​

Pivot To Center툴은 오브젝트의 중심에서 상하 위치를 빠르게 변경합니다.

Top Center​

선택 후 Top Center를 클릭하면 상단에 Pivot위치가 변경이 됩니다.

Center​

선택 후 Center를 클릭하면 중심에 Pivot위치가 변경이 됩니다.

Bottom Center​

오브젝트 선택 후 Bottom Center를 클릭하면 하단에 Pivot위치가 변경이 됩니다.
