Pipe Tool

The Pipe tool is a tool that creates a cylinder with a hole in the center.

How to use​

The basic shape creation tools in the Primitive Shapes group can create shapes in two ways.

  1. click-drag to create, edit
  2. Create and edit using One Click Build

Create and edit by clicking and dragging​

  1. Select the Pipe tool.

  2. Click and drag the mouse in the scene to create the outside of the base with the desired outside radius size.

  3. Once the shape is created at the desired size, release the click and move the mouse to the inside of the shape to generate a mesh at the desired inside radius size.

    note

    If you move the mouse outward rather than inward, the initially created radius will be inward and the area you move the mouse over will be outward.

  4. when the desired bottom is reached, click again and move the mouse up to generate a mesh at the desired height.

  5. When the desired height is reached, click again to finish creating the mesh.

  6. Adjust the properties of the Pipe tool and select the element, or press another tool to finish.

note

You can continue to refine the properties of the Pipe tool until you select an element or press another tool to finalize.

One Click Build and Edit via Properties​

  1. Select the Pipe tool.
  2. Enter the desired value in the Pipe tool's properties.
  3. Press the One Click Build button to add a shape to the scene.
  4. Make any additional adjustments to the Pipe tool's properties and select the element, or press another tool to finish.
note

You can continue to tweak the properties of the Pipe tool until you select an element or press another tool to finalize.

Properties​

Radius1​

This property sets the radius size of the shape being created based on its center.

When creating via Click-Drag, this is the property at the time of the first Click.

Radius2​

This property sets the radius size of the shape being created based on its center.

When creating via Click-Drag, this is the property on the second Click.

Height​

This property sets the height of the generated shape.

When creating via Click-Drag, this property is set on the third Click.

Sides​

Property that sets the number of pieces the perimeter surface of the generated shape will be split into.

Height Segment​

Property that sets the number of segments for the height of the generated shape.
