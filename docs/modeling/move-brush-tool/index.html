Move Brush Tool

The Move Brush tool is a brush-like tool that pulls on a vertex.

How to Use​

  1. Launch the Move Brush tool.
  2. adjust the desired properties.
  3. Move the mouse cursor over a vertex on the object, then click and drag to pull the vertex.

Properties​

Brush Radius​

This property determines the size of the brush's radius. The unit is meters.

tip

Use the keyboard shortcuts [ , ] , Ctrl+click-drag​

Hotkeys allow you to increase or decrease the Brush Radius property.

Brush Strength​

Property for the brush strength.

Fall Off​

A Graph Shape property that determines the shape of the brush.

It can be edited by clicking on the graph next to Fall Off.
