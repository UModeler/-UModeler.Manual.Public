Toolbar

UModelerX features a toolbar integrated with the Unity UI. Use this toolbar when you want to quickly adjust settings and check the results while working.

note

From Unity 2020.1 to 2021.1, the toolbar button is implemented to the right of the Tools button instead of the SceneView toolbar.

Display Settings​

All display settings can be easily configured by this toolbar. For more details, see Display Settings.

Snap Setting​

Use this toolbar to adjust the Snap Intervals for translation and rotation. For more details, see Snap Setting.

Coordinate System​

UModelerX includes a total of Five Coordinate Systems, including Unity's coordinate system. For more details, see Coordinate System.
