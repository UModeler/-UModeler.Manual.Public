Layer 그룹

페인팅 모드의 Layer 그룹은 레이어를 관리하고 편집할 수 있는 툴과 레이어가 표시된 그룹입니다.

정보

레이어(Layer)​

레이어는 개별적으로 편집이 가능한, 이미지들의 층으로 구성된 것을 말합니다. 레이어들은 최상단에 있는 레이어부터 아래쪽으로 겹치게 되며, 각 레이어의 영향은 블렌딩 모드라는 공식에 의해 계산되어 최종적으로 하나의 이미지로 표현됩니다.

인터페이스​

레이어 활성화​

레이어의 활성화 여부를 설정하는 속성입니다.

레이어 블렌딩 모드​

블렌딩 모드는 위에 있는 레이어가 아래에 있는 다른 레이어의 결과와 어떻게 혼합될지 결정하는 속성입니다.

레이어 불투명도(Opacity)​

레이어 오퍼시티는 위에 있는 레이어가 아래에 있는 레이어를 덮는 수준을 퍼센트로 조절할 수 있는 속성입니다.

레이어 맵소스별 표시​

레이어를 맵소스별로 표시를 하는 속성입니다. 선택한 맵소스에 따라 맵소스별로 레이어의 블렌딩 모드와 레이어 오퍼시티를 확인하고 편집할 수 있습니다.

Group 레이어​

Group 레이어는 레이어를 통합해서 관리하는 속성의 레이어입니다.

레이어 마스크​

레이어 마스크는 레이어의 특정 영역을 마스킹하는 속성입니다.

Dilation​

Dilation은 선택한 Paint 레이어의 칠해진 값이 UV 경계에 닿지 않았을 때, 해당 경계를 넘어서도 칠해진 값들을 확장시켜주는 기능입니다.

레이어 추가​

여러 속성의 레이어를 추가하는 버튼입니다.

Paint 레이어​

Paint 레이어는 칠하는 방식으로 맵소스를 적용하는 레이어입니다.

Fill 레이어​

Fill 레이어는 맵소스의 값 자체를 모델 전체에 덮는 방식의 레이어 입니다.

Sticker 레이어​

Sticker 레이어는 이미지를 표면에 붙이듯이 표현하는 속성의 레이어입니다.

레이어 삭제​

선택한 레이어를 삭제하는 버튼입니다.
