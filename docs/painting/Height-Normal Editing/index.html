Height-Normal Editing

Height-Normal Editing is a type of map source that enhances surface detail by applying height and normal information to give a sense of depth to the mesh surface.

How to use it​

  1. In the Map Source group properties, click Add Map Source > Height.

  2. A Height Map Source will be added. Check Height as Normal.

  3. Change the Material Mode to Preview to see the Height-Normal applied in real-time.

Properties Group Properties​

Texture​

This feature allows you to customize the brush by applying textures.

Value​

Determines the height value of the Brush. Based on a Background Value of 0.5, values closer to 0 will appear as engraving, while values closer to 1 will appear as embossing.

Height Map Source Group Properties​

Texture​

Sets the height of the background.

Value​

Adjusts the intensity of slopes when generating a normal map from height map information.
