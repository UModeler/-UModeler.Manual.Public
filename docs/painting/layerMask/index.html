Layer Mask

A layer mask is a property that masks a specific area of a layer.

How to use it​

  1. select the layer you want to add a layer mask to.
  2. Click the Layer Mask button.
  3. Select either Add Black Mask or Add White Mask.
  4. A layer mask has been added to the selected layer.

Properties​

Add Black Mask​

This property adds a black layer mask. As a result, all materials on the assigned layer will be hidden.

Add White Mask​

Property that adds a white layer mask.

Add AO Mask​

Property that adds an Ambient Occlusion (AO) layer mask.

Remove Mask​

Properties to remove the selected layer mask.

Layer Mask Properties Group Properties​

The Properties group is an area where the content varies depending on the feature selected, such as a tool or layer.

The properties displayed in the Properties group when Layer Mask is selected are shown below.

  • Brush
  • Erase
  • Rect Fill
Properties Tab

Brush

The Brush tool is a tool that allows you to apply mask values by painting.

ScreenSizeBrush​

Property that sets the brush size of the Brush tool in pixels, which when unset sets the brush size relative to its size on the world.

You can set the size in the Size property.

Size​

Property that sets the size of the brush.

Opacity​

This property allows you to set the opacity of the mask value being painted.

Hardness​

Property that allows you to set how sharp the edges of the brush in the Brush Tool are.

Jittering​

This property sets the amount of randomization of the strokes, which are the lines created by the brush input of the Brush Tool.

Spacing​

Property that allows you to set the spacing of the brush strokes generated by the Brush tool.

Mask Value​

This property allows you to set the mask value that will be painted.

Properties Tab

Erase

The Erase tool is a tool that allows you to remove painted mask values.

ScreenSizeBrush​

Property that sets the size of the brush for the Erase tool in pixels, which when turned off sets the brush size relative to its size on the world.

You can set the size in the Size property.

Size​

Property that sets the size of the brush for the Erase tool.

Opacity​

This property allows you to adjust the opacity of the erasure.

Hardness​

This property allows you to set how sharp the edges of the Erase Tool's brush are.

Jittering​

This property sets the amount of randomization of the strokes produced by the Erase Tool's brush input.

Spacing​

This property allows you to set how far apart the brush input of the Erase Tool is spaced.

Property Tab

Rect Fill

The Rect Fill tool is a tool that allows you to apply a mask value to a face via the Mode property.

Mode​

Properties for how the **Rect Fill tool works.

Polygon Fill

Tool that allows you to apply a mask value to a face when you click on the face.

Rect Fill

A tool that allows you to create a rectangular region by clicking and dragging in the scene, and apply a mask value to the faces that fall within it.

Polygonal Lasso Fill

This tool creates a polygonal region along vertices by clicking in the scene, and applies a mask value to the faces that fall within it.

Island Fill

A tool used with the Painting 2D Editor that allows you to click on a UV island in the Painting 2D Editor and apply a mask value to it.

Isolation Fill

A tool used with the Painting 2D Editor that allows you to apply a mask value to all connected elements by clicking on a UV face in the Painting 2D Editor.

Mask Value​

A property that allows you to set the mask value to be applied.

Camera Based Select​

A property that applies a mask value to only the currently visible face based on the camera.

TriangleMode​

This property applies the mask value to the triangular mesh that makes up the face.

Erase​

This property deletes the applied mask value.

Opacity​

A property that allows you to set the opacity of the applied mask value.
