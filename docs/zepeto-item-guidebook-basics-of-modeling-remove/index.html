Remove

The Remove group is a collection of tools that, when executed, will remove each element (Vertex, Edge, Face).

It mainly contains tools that either remove selected elements or merge elements by the tool.

Detach​

Detach separates selected faces or elements, either within the same object or as a new object.

Holding the Alt key and clicking Detach allows you to choose the Type.

In "To Element" mode, Detach will separate the face, but it will remain within the same object.

In "To Object" mode, Detach will create a new object in the Hierarchy, and you will see that the selected face has been separated into this new object.

Eraser​

Eraser deletes selected elements. You can also use the delete key for quick deletion.

Targer Weld​

Target Weld connects selected vertices to each other.

Collape​

Collapse merges selected elements into one. Note that this works only for connected groups.

Merge​

Merge combines selected vertices into one. Depending on the options, the merge can occur at different points between the selected vertices.

Weld​

Weld joins vertices within a specified distance of each other.

Combine​

Combine Face merges selected faces into one. Note that this may not work well with heavily curved or N-gon shapes.
