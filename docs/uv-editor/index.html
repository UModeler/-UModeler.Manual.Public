UV Editor

A UV is a texture coordinate in 2D form that corresponds to an object in 3D form. UV Editor allows you to create and edit UV elements (UV Vertex, UV Edge, UV Face).

How to Run​

Here's how to run the UV Editor.

Source Mesh>UV Editor Tool​

  1. Select a Source Mesh.
  2. Press the Editor button at the top right of the mode.
  3. Launch the UV Editor tool from the menu.
  4. You will see the following in the UV Editor.

Select an object>Select the open UV Editor tab​

  1. Select the UModeler X object.
  2. Click on the UV Editor tab among the open tabs to activate it.
    note

    If the UModeler X Object is not selected, the UV Editor will not be activated.

Properties​

The Properties of the UV Editor can be divided into four main areas, each of which is briefly described below.

1. Toolbar​

The following are listed in order: Screen Pan, Transform Gizmo, 3 UV Elements, UV Island Selection, Cursor, and Pivot Toggle.

2. Settings​

This is the area where you can make settings related to the UV Editor.

3. Working Area​

This is the area where you edit the UV element. You can see the texture along with the grid.

4. UV Tools​

This is the area with tools to select, edit, or export UVs.

Unwrap Group​

This group contains tools for unwrapping UV elements.

Selection group​

This group contains tools for selecting UV elements.

Quick Transform group​

This group contains tools that allow you to easily flip and rotate the position of UV elements.

Alignment group​

This group contains tools for aligning the position of UV elements.

Weld And Break group​

This group contains tools used to break and weld UV elements together.

Arrange group​

This group contains tools for placing UV elements in texture space in bulk.

Misc group​

This group contains tools that do not belong in any other group.
