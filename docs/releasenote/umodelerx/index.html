UModeler X Basic​

note

The UModeler X Beta version has been renamed to UModeler X Basic as of September 13, 2024.

Ver 1.0.3 | December 19, 2024​

  • Improvements
    • General
      • Improved performance issues caused by byte streams appearing in the inspector when selecting saved data.
      • Fixed issues occurring after VRChat uploads.
    • Rigging Mode
      • Envelope Tool: Capsules are not displayed for bones above the Root level.
    • Miscellaneous
      • The existing Line Tool has been moved to Add Group and renamed Knife Tool.
  • Bugs
    • General
      • Fixed an issue where some data such as UV Seam, Face Color, Face Normal, and UV Param were not saved when copying objects or saving scenes.
    • Modeling Mode
      • Face Tool: Fixed a bug where normal values were sometimes calculated incorrectly in the Smoothing Group.
    • Rigging Mode
      • Envelope Tool
        • Fixed an issue where the Show Handler feature did not work properly.
        • Capsule range is now displayed correctly according to the bone.
        • Fixed an issue where capsule size adjustments were not applied during Undo/Redo operations.
    • UV Editor
      • UV Face Tool: Fixed an issue where residual loops remained after canceling UV selection.

Ver 1.0.2 | November 5, 2024​

  • Improvements
    • General
      • Fixed an error that occurred when opening the UModelerX Editable Mesh in Unity’s Prefab Instance UI.
      • Clear Unused Assets: Addressed issues where the tool accessed non-existent file paths.
    • Modeling Mode
      • Vertex Tool: Double-clicking a vertex without pressing any keys now selects all connected vertices (Isolation Select Tool).
      • Bevel Tool: Maximum number of segments increased to 64.
      • Connect Tool: Edges created immediately after tool use are now automatically selected.
    • Rigging Mode
      • Weight Tool: Double-clicking a vertex without pressing any keys now selects all connected vertices (Isolation Select Tool).
      • Build Bone Tool: Improved bone head positioning during creation.
  • Bug Fixes
    • General
      • Fixed an error when opening UModelerX Editable Mesh in Unity’s Prefab Instance UI.
      • Clear Unused Assets
        • Resolved issues accessing non-existent file paths.
        • Fixed functionality issues when scene files were not in the Scene folder.
        • Corrected behavior when multiple scenes were open in the editor, applying changes only to the active scene.
      • Resolved issues with loading older UModelerX versions.
      • Preference: Fixed a bug where the reset function did not work correctly.
      • Resources: Removed warning messages related to icon resources.
      • Shortcut: Fixed issues where the UV Editor’s Weld and Break functions did not appear in Shortcuts.
      • Fixed a bug where some polygons temporarily disappeared when duplicating high-poly UModelerX Editable Meshes.
      • Fixed a bug with the Keep Quad option in the FBX export settings not working properly with multi-material models.
      • Resolved an issue in Unity versions 6.0 and above where overlapping model names caused some data to disappear when saving scenes.
    • Modeling Mode
      • Element Tool
        • Fixed bugs where Gizmos moved incorrectly with the Pivot Handle Position during extrusions.
        • Corrected inaccuracies in rotation direction when using local rotation.
        • Fixed an issue where entering "0" in the Distance field prevented further modification.
        • Enabled negative values in the Distance field.
        • Fixed an issue where the Distance value would not change continuously during handle manipulation.
        • Resolved a bug where the pivot center did not match the selection order.
      • Custom Normal Tool: Fixed a bug causing errors during data loading.
      • Connect Tool: Fixed an error occurring with Soft Selection enabled and using Undo after the Connect Tool.
      • Material Tool: Resolved an issue where adding new materials would mix existing face materials.
      • Face Normal Tool: Fixed issues with Undo/Redo functionality not working correctly after tool use.
      • Bridge Tool: Fixed an issue where selected elements remained after completing the bridge operation.
      • Local Settings Tool: Enabled instant updates when Recalculate Tangents is on.
    • Rigging Mode
      • Improved Weight value sorting during rigging builds.
      • Weight Tool: Fixed an issue where rigging weights displayed incorrectly on vertex selection.
      • Weight Tool: Fixed an issue where Undo/Redo did not update values in the Editor Inspector.
      • Reset Envelop: Fixed a bug where the Reset Envelop button was not functioning.
    • MultiObject Editor
      • Combine Tool: Fixed a bug where the tool did not work.
      • Material Tool: Fixed an issue where the Redo functionality did not apply material changes to multiple objects.
      • Local Settings Tool: Enabled instant updates when Recalculate Tangents is on.
    • Painting Mode
      • Fixed an issue with mask channel selection when changing layers.

    • UV Editor

    • Painting2D Editor
      • Fixed an issue that occurred with wireframe display.
    • MultiObjectEditor
      • Boolean Tool: Fixed an error that occurred when performing a Boolean operation between meshes that were not unwrapped.
      • Combine Tool: Fixed an issue where child objects would disappear when combining a GameObject with children.

0.20.0 | July 29, 2024​

  • New Features
    • Face Color Tool: A tool to apply different colors to adjacent faces sharing vertices.
      • Apply Color: Applies the selected color to the chosen faces.
      • Select Face Colors: Selects all faces with set face colors.
      • Reset Color: Removes the face color.
    • UVParameter Tool: A tool to modify automatically generated UVs for unwrapped faces.
      • Shift: Moves the auto-generated UV coordinates by the set value.
      • Rotation: Rotates the auto-generated UV coordinates around the origin, affecting the rotation center if the generation is offset.
      • Scale: Adjusts the scale of the auto-generated UV coordinates.
      • Tile: Sets tiled UV parameters for the selected faces, adjusting Shift, Rotation, and Scale values.
      • Select UVParams: Selects all faces with set UV parameters.
      • Reset UVParam: Removes the UV parameter settings.
  • Improvements
    • Element Tool: Now converts and selects elements based on previously selected vertices, edges, or faces when switching tools.
    • Overlay: Changed overlay rendering, fixing edge rendering issues.
  • Bug Fixes
    • Mirror Modifier: Fixed the issue where the arrow part of the move gizmo couldn't be picked.
    • Mirror Modifier: Fixed the problem where the move gizmo acted in the opposite direction when using mirror flip.
    • Connect Tool: Fixed the bug where commentary was not displayed.

  • Resolved an issue where after deleting a layer or group and undoing, it would not be recreated in its original location.
  • In Rigging mode, the Brush Radius Distance and Brush Radius Pixel values have been modified to not take negative values.
  • Fixed an issue where the bone and its name were not displayed when selecting the first bone in Rigging mode.
  • Fixed an issue that occurred when changing the bone used in Rigging mode.

  • Fixed an error that occurred when undoing/redoing after hot reloading in Rigging mode.
  • An option to display weight color has been added in Rigging mode. (Preferences > EdMesh > Rigging Weight RGB Color)

    • Mix: Adds or subtracts values until the set weight value is reached.
    • Add: Adds weight until it reaches 1.0.
    • Subtract: Subtracts weight until it reaches 0.0.
    • Blur: Smoothly interpolates between adjacent vertex weight values.
    • Lighten: Adds to values below the set weight until they reach the weight value (implementation planned).
    • Darken: Subtracts from values above the set weight until they reach the weight value (implementation planned).
