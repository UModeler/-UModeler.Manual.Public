Sew Tool

The Sew Tool in the UV Editor is used to sew selected UV edges to shared UV edges.

note

For a Shared Element​

A shared element is an element that is separated by a UV, but is attached or adjacent to it in the mesh.

UV Color

Shared elements are colored blue (default).

How to Use​

  1. Select the UV edges you want to connect.
  2. Run the Sew Tool.
  3. The selected UV edges will be moved to the shared UV edge and joined together.
