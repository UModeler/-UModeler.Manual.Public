Flip Horizontal Tool

The Flip Horizontal Tool in the UV Editor flips the position of the selected UV element relative to its center.

If the UV Editor's cursor is active, the position of the cursor will be centered and flipped.

note

For the UV Editor Cursor​

A point-like cursor used in the UV Editor when you want to use a tool relative to a specific location.

You can move the cursor's center point by Click-Drag it.

How to Use​

  1. Select a UV element.
  2. Run the Flip Horizontal Tool.
  3. The selected UV element is flipped left and right.
note

When Cursor is enabled​

If the cursor is active, it will be flipped left and right to center the position of the cursor.
