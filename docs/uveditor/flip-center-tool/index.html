Flip Center Tool

The Flip Center Tool in the UV Editor is a tool that flips the position up and down and left and right relative to the center of the selected UV element, resulting in a diagonal flip.

If the UV Editor's cursor is active, it will center flip the position of the cursor.

note

For the UV Editor Cursor​

A point-like cursor used in the UV Editor when you want to use a tool relative to a specific location.

You can move the cursor's center point by Click-Drag it.

How to Use​

  1. select a UV element.
  2. Run the Flip Center Tool.
  3. The selected UV element is centered flipped.
note

When Cursor is enabled​

If the cursor is active, it will be center flipped to center the position of the cursor.
