Shared Tool

The Shared tool in the UV Editor selects UV elements that are shared with the selected UV.

note

For a Shared Element​

A shared element is an element that is separated by a UV, but is attached or adjacent to it in the mesh.

UV Color

Shared elements are colored blue (default).

How to use​

Vertices​

  1. Select a vertex.
  2. Run the Shared tool.
  3. A vertex that shares a mesh with the selected vertex is selected.

Edges​

  1. Select an edge.
  2. Run the Shared tool.
  3. Edges that share a mesh with the selected edge are selected.
