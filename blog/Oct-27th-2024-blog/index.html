UModeler X Achievements and News - October 2024

October 27, 2024 · 5 min read
Jaesik Hwang
Head of UModeler Development

Hello everyone,

We're excited to share some thrilling updates about UModeler from October 2024 with our community.

Unite 2024 Participation​

UModeler participated as a sponsor at Unite 2024 (September 18-20, 2023), drawing significant attention from developers worldwide. Notably, we received exceptional interest not only from game developers but also from university educators and students, highlighting how in-Unity 3D modeling capabilities can significantly lower the barriers to education and learning.

UModeler X Plus Early Bird Reaches #1 in Popularity​

A remarkable achievement was the 'UModeler X Plus Early Bird' version reaching #1 in overall popularity on the Unity Asset Store just two days after its release on September 14th. This incredible success demonstrates developers' high expectations and trust in UModeler X Plus.

Continuous Evolution: 7 Significant Updates​

UModeler X Plus Early Bird has undergone remarkable development through seven updates, from 0.91.1 to 0.92.0. Let's look at the key updates.

Core New Features​

  • Align Tool: Precisely align model vertices according to various criteria

  • Normal Painting Tool: Advanced functionality for converting height maps to normal maps

Usability Improvements​

  • Enhanced Tablet Support: Automatic brush opacity and size adjustment based on pressure sensitivity
  • Clear Unused Assets: Efficient scene data cleanup
  • Primitive Shapes: Automatic object naming functionality
  • Vertex Position Editor: Precise position editing in local/world coordinate systems

Performance and Stability​

  • Improved compatibility with Unity 6.0+
  • Multi-material handling optimization
  • Enhanced UV and normal calculation accuracy
  • Various bug fixes and performance improvements

For more detailed information, please check the UModeler X Plus Release Notes.

UModeler X Plus 1.0​

UModeler X Plus 1.0's official release is scheduled for Wednesday, November 20th. As previewed in September, it will feature significantly upgraded capabilities.

Enhanced Modeling System​

  • Upgraded SketchUp Style Modeling: Continuous improvements for intuitive modeling workflow
  • Follow Tool: Enhanced power with progressive generation capabilities added to the existing robust Follow tool
  • Hotspot Texturing: Improved UI/UX and trim sheet support for enhanced texturing experience

System Improvements​

  • Enhanced Save System: Simultaneous .asset and .unity scene file saving for lighter project management
  • Humanoid Avatar Support: Complete Humanoid Avatar support in rigging mode for streamlined animation application

User-Centric License Policy Improvements​

After careful consideration of user feedback, UModeler has enhanced its licensing policy:

Basic Version​

  • Free for both commercial and non-commercial use
  • Updates focused on major bug fixes
  • Internet connection required
  • User data collection for service improvement
  • Current feature set maintained

Plus Version (After 1.0 Release)​

  • Premium license supporting both commercial and non-commercial use
  • Rapid updates and priority feedback implementation
  • Offline usage available
  • Optional user data collection
  • Advanced features including SketchUp-style modeling, Hotspot texturing, Array tool/modifier, Follow tool, Align tool, and Normal painting
