UModeler X 2025 New Year Update

January 3, 2025 · 5 min read
Jaesik Hwang
Head of UModeler Development

Hello everyone! Welcome to 2025! Today, we're excited to share the latest updates to UModeler X.

Enhanced UModeler X Plus Features​

This update introduces several new features designed to boost your productivity. Notable additions to the Unity toolbar include two new popup windows: UMX Display Settings and UMX Snap Settings. These windows eliminate the need to navigate through multiple menus to adjust display and snap settings during your workflow!

For those who love symmetrical modeling, we've enhanced the Mirror Object tool with a preview feature that shows the mirrored parts before creation, making symmetrical modeling more intuitive and efficient.

Character artists will appreciate our new Create Root Bone Structure feature, which automatically generates animation root bones and geometry object hierarchies for rigging. Say goodbye to complex manual setup before rigging!

We're also introducing the Element ID Selector, a powerful troubleshooting tool that visually displays IDs for vertices, edges, and faces, helping you quickly identify and resolve modeling issues.

Element ID Selector

Go to Documentation

Repetitive tasks just got easier with our new repeat function - simply press the `;` key to repeat your last action. This works with various tools including Bridge, Duplicate, Eraser, and Bevel.

Repeat Function

Go to Documentation

Additionally, we've renamed the Line tool to Knife tool and moved it to the Add group. The Drawing mode now features a new Polyline tool, which is continuously being improved!

Pro License​

UModeler X Plus introduces over 20 new features compared to the Basic version. Moving forward, Plus will focus on stability improvements and performance optimization rather than new feature additions.

The upcoming Pro license will include all Plus features while continuously adding new functionality. Pro will be offered as a subscription model to ensure sustainable company operations and stable development of UModeler X. This model will enable us to provide regular updates, new feature development, and enhanced technical support.

If you’re looking to enhance your workflow quickly and efficiently, check out UModeler X Plus.

A Message to Our Users​

We acknowledge recent feedback regarding stability issues with UModeler X Plus, particularly with the Polyline and PushPull tools for SketchUp-style modeling. We've also noted concerns about errors when switching between game and edit modes with Domain Reloading disabled.

We sincerely apologize for not meeting your expectations regarding stability. Our development team is working diligently to address these issues, and we're committed to improving stability throughout January to ensure a smooth experience for all users.

We deeply appreciate your feedback and patience as we work to make UModeler X more reliable and stable.

UModeler X Game Jam Announcement!​

First, let's talk about the UModeler X Game Jam. This game jam, hosted in collaboration with Future Inspire Academy (FIA), focuses on creating 3D games using Unity. FIA is a non-profit organization dedicated to providing game development education in a fun and flexible manner. They offer Unity challenges, game creation tasks, mentoring, and a gamified leaderboard to support developers worldwide.

We sponsor this game jam. Participants will use UModeler X's rigging, painting, AI texturing, and modeling features to develop creative games.
