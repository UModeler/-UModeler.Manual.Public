UModeler X Plus Early Bird - Finally Released!

September 14, 2024 · 5 min read
Jaesik Hwang
Head of UModeler Development

Hello everyone,

I’m excited to share some great news with you today. UModeler X Plus has finally launched on the Asset Store! In this blog, we’ll cover some useful information about UModeler X Plus.

Introducing UModeler X Plus​

UModeler X Plus combines the strengths of the original UModeler and UModeler X Beta. This tool makes real-time 3D content creation within Unity easier and more efficient, becoming the go-to 3D asset creation solution for Unity developers.

New Features in the Plus Version​

UModeler X Plus offers exciting additional features that make 3D modeling and texturing even more convenient

  1. SketchUp Style Modeling
  2. Push/Pull Tool
  3. Hotspot Texturing Tool
  4. Follow Tool
  5. Array Modifier and Tools
  6. Normal Painting
  7. .OBJ, .STL, .PLY Exporter

Differences between Plus, Basic (Beta), and the Original UModeler​

Early Bird Version Info for September​

The current Early Bird Version (0.90) includes the following features

  • Array Modifier and Tool
  • .OBJ, .STL, .PLY Exporter
  • SketchUp Style Modeling and Push/Pull Tool (Beta)

Future Updates​

  • More features will be added progressively throughout October and November.
  • The full version (1.0) with all features will be released in November.
  • If you purchase the Early Bird version of UModeler X, it will automatically upgrade to UModeler X Plus at no additional cost once the full version is released.

Special Bonus Offer​

Complete the registration and verification process after purchasing UModeler X Plus to receive the following assets for free.

  • Isometric Low-Poly Room Asset (Available for immediate download)
  • Dystopian Building Asset (Available in October)

These bonus assets are provided through the UModeler Hub.

Version Information and Data Compatibility​

  • The existing UModeler X Beta is now renamed UModeler X Basic, but its features remain unchanged.
  • Only one version (Basic or Plus) can be installed per project.
  • All features will work smoothly when upgrading from Beta or Basic to Plus.
  • Data is fully compatible between UModeler X Basic and Plus.

What about the UModeler X Beta Version?​

The existing Beta version will be automatically updated to the Basic 1.0 version, and all features in the Beta will continue to be available for free.
